# Iteration review template

Use this after each meaningful revision.

```json
{
  "iteration": "iteration-N",
  "skill_name": "<skill-name>",
  "summary": {
    "what_improved": [],
    "what_regressed": [],
    "open_issues": []
  },
  "scores": {
    "trigger_clarity": 0,
    "workflow_coherence": 0,
    "output_usefulness": 0,
    "evidence_of_real_leverage": 0,
    "overfitting_risk": 0,
    "evaluation_quality": 0
  },
  "benchmark_notes": [],
  "next_actions": []
}
```

Keep it short, concrete, and comparative.
Do not write generic praise.
