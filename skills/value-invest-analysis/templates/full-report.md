### 财务触发（季度跟踪，看财报后核对）
- ☐ {关键季度指标 1} 连续两季跌破 {threshold} → **重估并视情况减仓**
  - 行业示例：banking → NIM ≤ 1.5% / 不良率 > 2.0%；biotech → 现金跑道 < 18 月；real_estate → 三道红线全踩红
- ☐ FCF 连续两季为负且无明确解释（非战略 capex 周期）→ **立即重估**
- ☐ {行业专属指标}：{从 sector_extension 中加载，例如 banking → 拨备覆盖率 < 150% / insurance → 偿付能力综合充足率 < 150% / cyclical_commodity → 自由现金流跌破长协价对应的盈亏平衡}
- ☐ 应收账款周转天数 / 存货周转天数 显著恶化（≥ 50% YoY 上升）→ 重估并启动深度审计

### 事件触发（即时监控，发生即评估）
- ☐ **CEO / CFO / 创始人辞职 / 关键人物突逝**（特别是 `creator_led` 公司）→ **24 小时内重估**
- ☐ **立案调查 / 财报重述（Restatement） / 审计机构辞任** → **立即清仓**
- ☐ **主要做空机构发布做空报告**（Hindenburg / Muddy Waters / Citron / Wolfpack 等）→ **24 小时内决策**
- ☐ **重大监管反转**（如教培 2021 模式 / 蚂蚁 IPO 暂停模式 / 互联网反垄断 2021 模式）→ 立即评估"是否归零情景"
- ☐ {行业专属事件}：{从 sector_extension 加载，例如 banking → 大额坏账暴露 / biotech → 关键 III 期试验失败 / real_estate → 暴雷新闻 / VIE → 美国 SEC HFCAA 退市倒计时进入最后 1 年}
- ☐ {modifier_flags 专属}：
  - `vie_structure` → VIE 协议被法院判决无效 / 政策表态质疑
  - `pcaob_risk` → PCAOB 审计被拒
  - `a_h_dual_listed` → AH 折溢价持续异常 ≥ 60% 且无业务变化

**判定纪律**：以下情形 **不构成卖出理由**——
- 单纯股价下跌（除非伴随基本面恶化）
- 持有时间过长
- 大盘指数回调
- 短期市场情绪
- 业绩单季度不及预期但仍在指引区间内
