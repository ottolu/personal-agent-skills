# Sector: 高端消费品牌 (`consumer_brand_premium`)

**典型公司**：
- A 股：贵州茅台（兼伟大公司）、爱玛科技（部分品类）、片仔癀
- 港股：周大福 1929.HK、海伦司（早期）
- 美股：LVMH（巴黎）、Estée Lauder EL、Hermès（巴黎）、Ferrari RACE、Birkenstock BIRK

**v2 schema_extension**：无（base schema）
**主流派路由**：巴菲特（喜诗糖果传统） / 段永平

## 一、公司原型识别信号

- 品牌 ≥ 50 年历史 / 文化稀缺性 / 工艺壁垒
- 售价显著高于同品类均价（≥ 3× 均价为奢侈品）
- 毛利率 ≥ 60%（奢侈品 70%+）
- 客户为价格不敏感的高净值人群 / 收藏家
- 限量 / 工艺约束 / 配额制造稀缺
- 品牌跨代传承（祖辈 / 父辈 / 子辈连续消费）

## 二、商业模式六维适配

- **价值主张**：身份认同 + 工艺 + 稀缺性（不是"性价比"）
- **收入结构**：高毛利 / 低周转 / 全球分销
- **成本结构**：原材料相对低，但营销 / 门店 / 设计成本高；可控杠杆扩张
- **客户结构**：金字塔顶端 1-10% 人群；DTC 占比上升趋势
- **再投资循环**：精品店扩张 + 限量产品线扩张 + 收购小众品牌
- **关键资源**：品牌历史 / 工艺师 / 旗舰门店 / 全球分销网

## 三、护城河重点

最适用：**品牌（无形资产 / 文化稀缺）**

锚点：
- 10 分：Hermès（爱马仕 Birkin 配货制度 + 工艺师培训 10 年 + 跨代传承）、茅台
- 7 分：LVMH 旗下 LV、Estée Lauder 高端线（Tom Ford、La Mer）
- 验证：(a) 涨价后销量不降反升（吉芬品现象）；(b) 二手价 ≥ 原价（爱马仕、劳力士）；(c) 历史穿越多个经济周期

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 毛利率 | ≥ 65%（奢侈品 70%+） | < 50% |
| 营业利润率 | ≥ 25% | < 15% |
| 同店销售增长（SSSG） | ≥ 5% | < 0% |
| 存货周转天数 | 180-365 天（奢侈品 OK） | > 500 天（积压） |
| 二手市场溢价率 | ≥ 80% 原价 | < 50%（品牌力弱化） |
| 中国 / 亚太收入占比 | 跟踪重要趋势 | 单一市场依赖 > 60% |
| DTC（直营）占比 | ≥ 70%（控价 + 利润最大化） | < 50% |
| ROIC | ≥ 20% | < 15% |
| 营销费用 / 营收 | 8-15%（合理） | > 25%（品牌靠投放维持） |
| 品牌组合多样性 | ≥ 5 个品牌（LVMH 型）或单一极致（爱马仕） | 单品牌且生命周期堪忧 |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F10（高杠杆）— 多净现金；F22（存货周转过慢）— 奢侈品可允许 365+ 天
- **改读项**：
  - F22 存货周转：奢侈品 200-400 天属正常，但若主品类 > 500 天则积压
  - F8 应收账款：DTC 模式下应收账款本应低，若飙升要追问
  - F26 频繁会计变更：奢侈品集团合并报告复杂，关注存货计价方式
- **新增类型专属红旗**：
  - CB1：水货 / 假货市场泛滥但公司无作为（品牌价值流失）
  - CB2：核心设计师 / 工艺师离任（爱马仕、香奈儿型公司极敏感）
  - CB3：明星代言 / 营销丑闻（D&G 辱华事件）
  - CB4：被并购入大集团但品牌定位被稀释（部分 LVMH 收购案例）

## 六、估值方法优先级

1. **DCF（10 年 + 永续）** — 主方法，永续增长率 2-3%
2. **EV/EBITDA + EV/Sales 倍数法** — 奢侈品行业普遍 EV/EBITDA 15-25×
3. **品牌价值评估法** — 参考 Interbrand 估值
4. **Reverse DCF** — 检验市场预期

关键假设：
- 不要给奢侈品过度永续增长（中国市场反腐 / 反炫耀消费风险）
- 历史 PE/EV 估值带是重要参考

## 七、同业对比策略

- **同业范围**：全球（LVMH vs Hermès vs Kering vs Richemont vs Estée Lauder vs L'Oréal）
- **同业数量**：3-5 家
- **对比维度**：毛利率、SSSG、中国占比、DTC 占比、ROIC、品牌组合稀释程度

## 八、关键风险与黑天鹅

1. 中国反腐 / 反炫耀消费政策（如 2013-2014 周期）
2. 经济衰退导致高净值消费下行（2008、2020）
3. 假货 / 灰色市场冲击
4. 创始人 / 设计师 / 工艺师传承断层
5. 年轻一代偏好转向（极简 / 二手 / 可持续消费）
6. 关税战 / 货币波动（美中、美欧）
7. 营销丑闻 / 品牌定位失误
8. 并购整合失败（被大集团收购后失去灵魂）

## 九、required_pct 默认值

`base = 25%`（V2_SPEC §六）

调整后典型：
- Hermès 级稀缺品牌：25% - 5% = **20%**
- 普通高端：**25%**
- 周期承压：**30%+**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（同店销售、品类拆分、地区拆分）
- 主要数据源 2：贝恩 / 麦肯锡 奢侈品年报（行业趋势）
- 主要数据源 3：StockX / 得物 / The RealReal 二手价格数据（品牌强度）
- 主要数据源 4：海关数据 / 出境游 / 免税店数据（中国占比）

## 十一、典型案例参考

**Hermès RMS**：爱马仕 Birkin 等核心产品采"配货制度"——客户须长期购买周边后才获购入资格。毛利率 70%+，ROIC 30%+，跨代传承数十年。风险：工艺师培训瓶颈、家族治理稳定性、过度依赖中国 / 美国市场。

**贵州茅台 600519.SH**：飞天茅台为中国白酒高端代表，渠道配额制造稀缺。商业模式同时具有 great_company_compounder + consumer_brand_premium 特性；主类型按"伟大公司"或"高端消费"二者均可。
