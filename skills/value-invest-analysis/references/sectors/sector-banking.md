# Sector: 商业银行 (`banking`)

**典型公司**：
- A 股：招商银行 600036.SH、宁波银行、工商银行、平安银行
- 港股：招商银行 3968.HK、汇丰控股 0005.HK、建设银行 0939.HK
- 美股：JPMorgan JPM、Bank of America BAC、Wells Fargo WFC、Capital One COF

**v2 schema_extension**：`banking.json`（NIM / 不良率 / 拨备覆盖率 / 核充率 / RWA / 零售贷款占比 / AUM / 中收占比 / 资本充足率分级）
**主流派路由**：巴菲特（招行、JPM 历史观察） / 巴菲特银行投资框架

**默认 modifier_flags**：必带 `regulated_heavily`；中资银行多带 `state_owned`；A+H 上市带 `a_h_dual_listed`

## 一、公司原型识别信号

- 主营吸收存款 + 发放贷款 + 中间业务
- 总资产规模通常 ≥ 万亿
- 监管核心指标：核心一级资本充足率、不良率、拨备覆盖率
- 净利息收入占比 ≥ 60%
- ROE 历史区间通常 10-18%（不同周期）
- 杠杆倍数 10-15×（金融机构属性，不可用工商 ROIC 评判）

## 二、商业模式六维适配

- **价值主张**：资金中介 + 风险管理 + 财富管理
- **收入结构**：净利息收入（NII）+ 手续费及佣金（中收）+ 投资收益
- **成本结构**：人力 + 网点 + 信贷成本（拨备）+ 监管合规
- **客户结构**：零售（个人）+ 对公（企业）；零售占比越高质量越高
- **再投资循环**：资本充足率约束下扩张 RWA；分红 + 留存补充资本
- **关键资源**：客户基础 / 牌照 / 资本 / 风控系统 / 网点 + 数字化能力

## 三、护城河重点

最适用：**牌照壁垒 + 客户切换成本 + 规模经济 + 低成本负债**

锚点：
- 10 分：（罕见）招行（零售 AUM + 私行 + 网银 + 客户体验综合）、JPM（IB + 零售 + 资管多元）
- 7 分：宁波银行（区域龙头）、宇宙行工行（规模）
- 验证：(a) 活期存款占比 ≥ 50%（低成本负债）；(b) NIM 高于行业 + 不良率低于行业；(c) AUM 增速 ≥ 资产增速（财富管理转型成功）

## 四、核心财务指标（替代通用指标，参考 schema_extension: banking.json）

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 净息差 NIM | ≥ 2.5%（中资）/ ≥ 3%（美资） | < 1.8% 或持续收窄 |
| 不良贷款率 NPL | ≤ 1.0% | ≥ 2.0% |
| 拨备覆盖率 | ≥ 300%（招行级） | ≤ 150%（监管红线 120%） |
| 核心一级资本充足率 CET1 | ≥ 10%（中资）/ ≥ 12%（美资） | ≤ 监管要求 + 1% |
| ROE | ≥ 15% | ≤ 10% |
| 成本收入比 C/I | ≤ 30% | ≥ 45% |
| 零售贷款占比 | ≥ 50% | ≤ 30% |
| 活期存款占比 | ≥ 50% | ≤ 35% |
| 非利息收入占比（中收） | ≥ 30% | ≤ 15% |
| AUM（财富管理规模）增长率 | ≥ 15% | ≤ 5% |
| 关注类贷款占比 | ≤ 2% | ≥ 4% |
| 逾期 90+ 与不良比 | ≤ 80%（确认充分） | ≥ 100% |

## 五、红旗清单 N/A / 改读规则

银行业 **N/A 红旗共 9 项**（财务结构完全不同于工商企业）：

- **N/A 项**：
  - F1（营收增速 < 行业）：N/A，应改用 RWA 增速、AUM 增速
  - F2（应收账款 > 营收）：N/A，应收账款不是银行核心概念
  - F3（毛利率定义）：N/A，银行用 NIM
  - F4（毛利率下降）：N/A，应改读 NIM 收窄
  - F5（库存 / 应收周转）：N/A，无库存
  - F6（连续亏损）：极少见，若发生为重大警告
  - F8（应收账款 > 营收增速）：N/A
  - F9（经营性现金流为负）：N/A，银行现金流定义不同
  - F12（资产负债率 > 70%）：N/A，银行杠杆 10-15× 是模型本身
  - F22（存货周转）：N/A

- **改读项**：
  - F7（ROIC 低）：用 ROE 替代，并看 ROA 是否 ≥ 1%
  - F10（杠杆经营依赖）：用核心一级资本充足率与监管比较
  - F19（现金枯竭）：用流动性覆盖率 LCR ≥ 100% 替代
  - F23（商誉占比高）：仍适用

- **新增类型专属红旗**：
  - BK1：不良率连续 2 季度上升 ≥ 30bps
  - BK2：核心一级资本充足率距监管要求 < 1%
  - BK3：拨备覆盖率从 300% 跌至 < 200%
  - BK4：零售贷款不良率突然飙升（信用卡 / 房贷分项）
  - BK5：关注类贷款占比 > 不良率 2 倍（潜在恶化）
  - BK6：大行对公敞口过高（如房地产、地方平台、城投）
  - BK7：净息差跌至历史最低且未见拐点
  - BK8：监管处罚 / 反洗钱违规 / 关键风险事件

## 六、估值方法优先级

1. **PB-ROE 估值（核心方法）** — Justified PB = (ROE - g) / (Ke - g)
2. **股息折现 DDM** — 高分红银行（汇丰、四大行）适用
3. **剩余收益模型 RIM** — 用账面价值 + 未来超额回报折现
4. **EVA / Economic Spread** — ROE - 资本成本
5. **历史 PB / PE 估值带** — 辅助参考

关键假设：
- Ke 取 10-12%（中资）/ 8-10%（美资）
- 永续 ROE 假设要保守（中资银行 12-15%，美资 10-13%）
- 不用 DCF / FCF —— 银行 FCF 概念不适用

## 七、同业对比策略

- **同业范围**：同市场 + 跨市场（招行 vs 宁波 / 平安 / 工行；招行 vs JPM / 美银 / Wells Fargo）
- **同业数量**：3-5 家
- **对比维度**：NIM、不良率、拨备、CET1、ROE、零售占比、中收占比、AUM 增长

## 八、关键风险与黑天鹅

1. 房地产 / 地方平台 / 城投敞口暴雷
2. 利率周期下行压缩 NIM
3. 净息差挤压（存款竞争、利率市场化）
4. 严监管 / 反洗钱罚款（汇丰、富国曾被巨额罚款）
5. 系统性金融危机（2008 / 2023 SVB 等）
6. 房贷违约率上升
7. 中概 PCAOB / VIE 风险（外资银行较少触及）
8. 资本市场动荡冲击 AUM 与中收

## 九、required_pct 默认值

`base = 25%`（V2_SPEC §六）

调整：
- 优秀（招行、JPM）：25% - 5% + 3（regulated_heavily）= **23%**
- 一般国有大行：25% + 0 + 3 = **28%**
- 困境银行（SVB、Credit Suisse 类前夜）：+10 → **38%+**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（IFRS9 / CECL 拨备分阶段、不良迁徙）
- 主要数据源 2：监管披露（银保监会 / FDIC / FED 监管报告、压力测试）
- 主要数据源 3：评级机构（Moody's / S&P / 中诚信）
- 主要数据源 4：央行 LPR / 美联储利率决议（NIM 预测）
- 主要数据源 5：30 大房企违约公告（地产敞口风险）

## 十一、典型案例参考

**招商银行 600036.SH / 3968.HK**：中国零售银行龙头，私行 AUM 超 10 万亿，活期占比超 60%，NIM 2%+，不良率 ~1%，拨备覆盖率 400%+。V1 报告给出 Watchlist 7.475；V2 应用银行专属指标 NIM / 不良 / 核充率替代失效的 ROIC，加载 banking schema_extension，财务红旗 N/A 9 项有明确依据。叠加 `[a_h_dual_listed, state_owned, regulated_heavily]`。

**JPMorgan Chase JPM**：美国宇宙行，投行 + 商业银行 + 资管 + 财富管理多元。Jamie Dimon CEO 长期主义经营，ROE 15%+，CET1 13%+。可作 management 锚点 7 分案例。
