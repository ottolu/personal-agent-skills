# Sector: 高速增长平台/电商 (`high_growth_platform`)

**典型公司**：
- A 股：无典型（A 股科技平台稀缺，仅有快手 1024.HK 偏港股）
- 港股：拼多多 PDD（美股）、快手 1024.HK、B 站（港美双重）
- 美股：Sea Limited SE、Shopify SHOP、MercadoLibre MELI、DoorDash DASH、Uber UBER

**v2 schema_extension**：无（base schema；可考虑 stage 2 补）
**主流派路由**：张磊（动态护城河 / 长期主义）

## 一、公司原型识别信号

- 营收 CAGR ≥ 30%（过去 3 年）
- 用户基础 / GMV 高速扩张但商业模式未完全稳态
- 仍有显著 SBC（≥ 10%）和大额营销投入
- 多线业务尚在尝试中（电商 + 金融 + 物流 + 直播 + 海外）
- 毛利率与 take rate 仍在动态调整
- 部分细分业务已盈利但整体盈利能力波动大

## 二、商业模式六维适配

- **价值主张**：颠覆现有龙头（性价比 / 即时配送 / 全球化）
- **收入结构**：佣金 + 广告 + 增值服务，可能尚未稳定
- **成本结构**：高获客 + 高补贴 + 高研发；规模效应正在显现
- **客户结构**：用户 + 商家双边，下沉市场 / 海外新增重要
- **再投资循环**：补贴扩张 + 物流 / 仓储 / 海外基建投入
- **关键资源**：用户增长能力 / 算法 / 资本（融资能力）

## 三、护城河重点

最适用：**网络效应 + 规模经济 + 数据飞轮（动态、未必稳固）**

注意：高增长不等于护城河成熟。张磊"动态护城河"概念适用 —— 评估护城河是在变宽还是变窄。

锚点：
- 7 分：拼多多（已建立社交电商护城河 + 农产品供应链）、Sea（东南亚电商 + 游戏双引擎）、MELI（拉美电商 + 物流 + 支付三引擎）
- 5 分：DoorDash / Uber（双边市场已建立但商业模型仍受挑战）

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 营收增长率 YoY | ≥ 30% | < 15% 且 take rate 持平 |
| GMV / 用户数 增速 | ≥ 30% | < 15% |
| Take Rate（货币化率） | 稳定上升 | 下降或停滞 |
| 单位经济模型 LTV/CAC | ≥ 3× | < 2× |
| 现金消耗率（gross burn） | < 1 年内能盈利 | > 18 个月仍亏 |
| Cash Runway | ≥ 24 个月 | < 12 个月 |
| Operating Margin | 转正且改善 | 持续恶化 |
| SBC / Revenue | ≤ 12% | > 20% |
| Free Cash Flow | 转正或趋势改善 | 持续大幅负 |
| ROIC | 难测，看趋势 | -- |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F6（连续亏损）— 早期亏损可接受；F7（ROIC 低）— 不适用；F10（杠杆）— 通常股权融资为主
- **改读项**：
  - F1（营收增速 ≪ 行业）：高增长平台必须远超行业，若与行业持平视为重大警告
  - F14（SBC 高）：≤ 15% 可接受；> 20% 视为重红旗
  - F8（应收账款）：电商平台预付为主，反向考察
- **新增类型专属红旗**：
  - HG1：增长依靠"红包 / 大额补贴"，撤补贴即丢用户（瑞幸早期模式）
  - HG2：LTV/CAC < 2× 仍在扩张
  - HG3：现金 runway < 12 个月且尚未盈利
  - HG4：监管 / 反垄断 / 数据合规专项整治（如 2021 滴滴 / 教培 / 平台经济）
  - HG5：主营业务面临成熟竞争对手反扑且份额开始下滑

## 六、估值方法优先级

1. **DCF（明确预测期 5-10 年 + Gordon）** — 主方法但敏感性巨大，必须做多情景
2. **EV/Sales 倍数法** — 行业可比公司估值带
3. **Rule of 40**（营收增速 + 利润率 ≥ 40%）— SaaS / 平台健康度判断
4. **Reverse DCF** — 极其重要，检验隐含增长是否过激
5. **Sum-of-Parts** — 多业务线公司（Sea = Garena + Shopee + SeaMoney）

关键假设：
- 永续增长率 3-4%
- 必须做 base / bull / bear 三情景 DCF
- WACC 9-12%（不确定性大）

## 七、同业对比策略

- **同业范围**：全球（拼多多 vs Sea vs MELI vs Shopify；DoorDash vs Uber Eats vs Meituan）
- **同业数量**：3-5 家
- **对比维度**：增长率、take rate、单位经济、operating margin、SBC 比例

## 八、关键风险与黑天鹅

1. 高估值挤泡沫（2021 → 2022 跌 70% 现象普遍）
2. 监管打压（反垄断 / 平台经济专项 / 数据出境）
3. 主业增长见顶 + 新业务尚未起量（断档期）
4. 单位经济恶化 / 持续烧钱无法盈利（部分共享经济）
5. 资本市场关闭融资（私募估值倒挂）
6. 创始人 / CEO 关键决策风险（黄峥退出 / 马云退出 / Adam Neumann）
7. 海外扩张失败（Shopee 拉美巴西黯然退出）
8. 中概股 VIE / PCAOB 风险（如适用）

## 九、required_pct 默认值

`base = 40%`（V2_SPEC §六，反映不确定性溢价）

调整：
- 优秀（拼多多 / MELI）：40% - 5% = **35%**
- 含 VIE + PCAOB：40% + 5 + 5 = **50%**
- 含 distressed：+10 → **50%+**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（GMV / Take Rate / Active Buyer）
- 主要数据源 2：App Annie / Sensor Tower / QuestMobile（用户运营指标）
- 主要数据源 3：竞对披露（用户数对比）
- 主要数据源 4：海关 / 出口 / 国际物流数据（跨境电商）

## 十一、典型案例参考

**拼多多 PDD**：从社交电商切入，下沉市场起家，Temu 全球扩张。2024 仍高增长 + 高盈利能力。风险：黄峥退出后治理透明度（management 锚点 3 分案例）、Temu 在欧美关税战、中概股 PCAOB / VIE 风险。可叠加 `[vie_structure, pcaob_risk, adr_listed]`。

**Sea Limited SE**：Garena 游戏 + Shopee 电商 + SeaMoney 金融。东南亚领先，但拉美 / 印度扩张烧钱受阻。2021 估值见顶后跌 80%。须用 SOTP 估值。
