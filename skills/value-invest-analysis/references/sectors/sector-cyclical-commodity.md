# Sector: 大宗商品周期股 (`cyclical_commodity`)

**典型公司**：
- A 股：中国神华 601088.SH、紫金矿业 601899.SH、中国石油、宝钢股份
- 港股：中国神华 1088.HK、中石油 0857.HK、紫金 2899.HK
- 美股：ExxonMobil XOM、Chevron CVX、Freeport-McMoRan FCX、Vale VALE、BHP BHP

**v2 schema_extension**：`cyclical_commodity.json`（commodity_price_history / normalized_eps / pb_cycle_percentile / 长协占比 / 自有运输 / 采矿成本曲线）
**主流派路由**：霍华德·马克斯（周期判断） / 巴菲特（西方石油案例）

**默认 modifier_flags**：中资多带 `state_owned`、`a_h_dual_listed`

## 一、公司原型识别信号

- 主营石油 / 天然气 / 煤炭 / 钢铁 / 铜 / 铁矿石 / 黄金 / 化肥 等大宗商品
- 营收与商品价格高度相关（弹性 1-2×）
- ROE 周期性极大（高点 25%+，低点 < 5% 甚至负）
- 资本密集（PP&E / 总资产 ≥ 50%）
- 净利润对大宗价格敏感（每 $10 油价变化 → EPS 变化 30-50%）
- 派息率与商品价格周期同步（高景气期高分红）

## 二、商业模式六维适配

- **价值主张**：提供工业基础原料 / 能源
- **收入结构**：商品销售（部分长协 + 部分现货）
- **成本结构**：开采 + 加工 + 运输 + 资源税 + 折旧；成本曲线决定 ROIC
- **客户结构**：电力 / 化工 / 钢厂 / 政府储备
- **再投资循环**：勘探 + 开发 + 收购；周期顶端易过度投资
- **关键资源**：储量 + 矿区品位 + 采矿权 + 运输配套

## 三、护城河重点

最适用：**低成本资源（成本曲线位置）+ 储量稀缺 + 一体化运输**

锚点：
- 7 分：神华（采运一体化 + 自有铁路 + 港口）、Saudi Aramco（成本曲线最左）
- 5 分：紫金（资源储量 + 全球化）、中石油（国内 + 海外多元）
- 3 分：成本曲线右侧的高成本煤炭 / 石油公司

验证：
- 成本曲线位置（最左 = 最强护城河）
- 周期低点是否仍能盈利
- 储量替代率（Reserve Replacement Ratio）≥ 100%

## 四、核心财务指标（参考 schema_extension: cyclical_commodity.json）

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 单位完全成本（C1 / AISC） | 成本曲线 1st quartile | 第 4 quartile（周期低点亏损） |
| 储量替代率 RRR | ≥ 100%（持续补充储量） | ≤ 70%（储量耗尽） |
| 储量寿命 | ≥ 15 年 | ≤ 8 年 |
| Normalized EPS（穿越周期均值） | 用 10 年均值，看当前价对应 PE | 现价 / 顶部 EPS 估值（陷阱） |
| PB Cycle Percentile | < 30%（周期低位）建仓 | > 70%（周期高位）回避 |
| 长协占比（煤炭 / LNG） | ≥ 60% | < 30% |
| 自有铁路 / 港口 / 管道占比 | ≥ 50% | 完全依赖外部物流 |
| ROE 历史区间 | 周期均值 ≥ 15% | 周期均值 < 8% |
| Net Debt / EBITDA | ≤ 1.5×（周期均值） | ≥ 3× |
| 分红率（高景气期） | ≥ 50% | < 30% |
| 资本开支 / OCF | ≤ 50%（克制）| > 80%（顶端过度投资） |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1（营收 / 行业）— 周期股营收随商品价格波动，需用 normalized；F7（ROIC 低）— 周期低点 ROIC 必然低
- **改读项**：
  - F6（连续亏损）：周期低点亏损可能但非红旗；若连续 5 年亏损则严重
  - F4（毛利率下降）：随商品价格波动正常，但毛利率持续低于成本曲线分位异常
  - F12（高负债）：与 EBITDA 比较，周期均值 Net Debt / EBITDA ≤ 1.5× 是关键
- **新增类型专属红旗**：
  - CC1：周期顶端进行大规模并购或新建（如 2008 钢铁、2014 油气、2021 锂矿顶部并购）
  - CC2：储量替代率 < 70% 连续 2 年（资源枯竭无补充）
  - CC3：单位完全成本上升至成本曲线第 4 quartile
  - CC4：政府提高资源税 / 出口关税 / 配额
  - CC5：ESG / 碳关税 / 转型政策导致传统资产搁浅
  - CC6：地缘政治（俄乌 / 中东冲突 / 关键运输通道封锁）
  - CC7：去库存周期（钢铁 / 煤炭 / 铁矿石客户大幅减库）

## 六、估值方法优先级

1. **Normalized PE / EV** — 用 10 年穿越周期均值 EPS，给合理 PE 12-15×
2. **PB Cycle Percentile** — 周期低位 PB 一般 < 1×，高位 PB > 2×
3. **DCF（商品价格三情景）** — Base / Bull / Bear 三套商品价假设
4. **Replacement Cost** — 重置成本估值（铜、铁矿、油气）
5. **Reserves × Price - Cost** — 储量资产估值法

关键假设：
- 永续增长率 0-2%（不应假设商品价持续上涨）
- 商品长期均价取过去 10-20 年通胀调整后均值
- WACC 9-12%（周期风险溢价）

## 七、同业对比策略

- **同业范围**：全球（煤炭 = 神华 vs Glencore vs Peabody；铜 = 紫金 vs Freeport vs Antofagasta；油 = 中石油 vs Exxon vs Saudi Aramco）
- **同业数量**：3-5 家
- **对比维度**：单位成本、储量寿命、负债率、长协占比、海外占比

## 八、关键风险与黑天鹅

1. 大宗商品价格暴跌（2014 油价、2015 铁矿石、2020 油价负值）
2. 周期顶端过度并购 / 扩产
3. 监管 / 资源税 / 出口配额
4. 地缘政治冲突
5. 环保 / ESG / 碳中和 / 转型搁浅资产
6. 重大安全事故（井喷、矿难、溃坝 —— Vale Brumadinho、BP 漏油）
7. 美元 / 汇率冲击（大宗商品美元计价）
8. 全球需求衰退（铜 / 钢与房地产 - 基建周期）

## 九、required_pct 默认值

`base = 35%`（V2_SPEC §六，反映周期不确定性）

调整：
- 低成本龙头（神华、Saudi Aramco）：35% - 5% = **30%**
- 周期顶端建议持仓更谨慎：+10 = **45%**
- 周期低位质量优秀（紫金 2015）：可降至 **25-30%**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（产量 / 售价 / 单位成本 / 储量）
- 主要数据源 2：商品价格数据（Bloomberg / Wind / IEA / IEEF / Wood Mackenzie）
- 主要数据源 3：行业协会数据（中国煤炭运销协会、CRU、SMM 上海有色网）
- 主要数据源 4：地质储量披露（JORC / NI 43-101 报告）
- 主要数据源 5：宏观（中国 PMI / 美国通胀 / 全球需求模型）

## 十一、典型案例参考

**中国神华 601088.SH / 1088.HK**：煤炭 + 自有铁路 + 港口 + 电力一体化龙头。成本曲线最左侧（每吨完全成本 < 250 元）。长协占比 ≥ 80%，盈利稳定性远超普通煤炭股。2023-2024 高分红率（≥ 70%），股息率 8%+。叠加 `[a_h_dual_listed, state_owned, with_policy_support]`。可作"周期 + 现金流稳定 + 高分红"复合典型，部分时段可归入 `declining_cash_cow`。

**ExxonMobil XOM**：全球油气龙头，垂直一体化（上游 + 中游 + 下游 + 化工）。穿越多个油价周期，2020 油价负值后强力回升。2024 PE 12×，分红率 40%+。
