# Sector: 衰退中现金牛 (`declining_cash_cow`)

**典型公司**：
- A 股：中国电信 / 联通（部分时段）、双汇发展、中国神华（部分时段）
- 港股：中国移动 0941.HK、中国海洋石油 0883.HK
- 美股：Altria MO（万宝路烟草）、IBM、AT&T T、Verizon VZ、Western Union WU、Kraft Heinz KHC

**v2 schema_extension**：无（base schema）
**主流派路由**：巴菲特（早年 Western Union 案例）/ 股息派 / 价值陷阱识别

## 一、公司原型识别信号

- 主业进入结构性衰退（年化营收下滑 1-5%）
- 但仍产生大量自由现金流（FCF Margin ≥ 15%）
- 资本开支需求低（capex/OCF < 30%）
- 高分红率（≥ 50% 甚至 100%+ Payout）
- 历史龙头地位 + 品牌 / 网络 / 客户基础仍存
- 股价反映"衰退故事"（PE 通常 < 10，股息率 ≥ 5%）

## 二、商业模式六维适配

- **价值主张**：仍满足某些细分需求（吸烟成瘾、传统电话、地面无线、传统媒体）
- **收入结构**：成熟产品销售 + 订阅 + 服务费
- **成本结构**：折旧 + 维护 + SG&A；变动成本低
- **客户结构**：忠诚但老化的客户基础
- **再投资循环**：低资本开支 → 高分红 / 回购 / 偿债
- **关键资源**：品牌 / 网络资产 / 客户基础 / 牌照

## 三、护城河重点

最适用：**品牌成瘾 / 切换成本 / 监管壁垒（衰减中）**

锚点：
- 7 分：Altria（烟草成瘾性 + 监管壁垒 + 全球高利润）、Microsoft（早期 Windows / Office 模式，现已转型）
- 5 分：中国移动（用户基础大但 ARPU 增长有限）、AT&T / Verizon（无线寡头）
- 3 分：IBM（云转型挣扎）、Kraft Heinz（消费品 OG 但被 PE / 自有品牌冲击）
- 0 分：已无现金流的衰退公司

验证：
- FCF 是否稳定 / 仅缓慢下降
- 分红是否可持续覆盖
- 衰退率是否在加速（vs 缓慢有序衰退）

## 四、核心财务指标

| 指标 | 优秀 / 健康 | 警戒 / 价值陷阱 |
|---|---|---|
| 营收增速 | -1% ~ +2%（缓慢下滑可接受） | ≤ -5%（加速衰退） |
| FCF Margin | ≥ 15% 稳定 | < 10% 或快速下滑 |
| FCF / Dividend（覆盖倍数） | ≥ 1.5× | ≤ 1×（不可持续） |
| 分红率（Payout） | 60-90% | > 100%（举债分红） |
| Net Debt / EBITDA | ≤ 3× | ≥ 4× |
| 利息覆盖 | ≥ 4× | ≤ 2× |
| Capex / OCF | ≤ 30% | ≥ 50%（仍要重投入） |
| 客户流失率 / Churn | < 5%（粘性强） | ≥ 10% |
| 业务转型进展 | 新业务 / 数字化收入占比 ≥ 20% 且增长 | 无转型路径 |
| 评级 | 投资级稳定 | 高收益级 |
| 历史股息连续年数 | ≥ 20 年（Dividend Aristocrat 候选） | 已削减分红 |
| 回购金额 / FCF | ≥ 30%（理性回购） | 高位回购 / 不理性 |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1（营收增速 ≪ 行业）— 衰退是定义本身；
- **改读项**：
  - F1（营收下滑）：从衰退率角度看，- 5% / 年是死亡螺旋警戒
  - F7（ROIC 低）：很多衰退 cash cow 仍有高 ROIC（如 Altria），不能机械套用
  - F10（高杠杆）：cash cow 用现金还债 / 维持低杠杆为佳
  - F12（资产负债率高）：与 EBITDA 比较，不机械套用
- **新增类型专属红旗**：
  - DC1：分红被削减（信号转折性恶化）
  - DC2：营收衰退率加速（- 1% → - 5%）
  - DC3：FCF / Dividend < 1×（举债分红不可持续）
  - DC4：核心产品被颠覆加速（如电子烟对传统烟草）
  - DC5：监管打压加剧（FDA 禁售薄荷烟、欧盟限烟）
  - DC6：评级下调（投资级 → 高收益）
  - DC7：业务转型尝试连续失败（IBM 多次战略转型挫败）
  - DC8：客户流失率加速（无线 / 有线电视退订）

## 六、估值方法优先级

1. **DDM 股息折现** — 主方法（高分红 + 稳定）
2. **FCF Yield + 衰退假设 DCF** — 用永续衰退率（- 1% ~ - 3%）
3. **历史 PE 估值带** — 通常 PE 7-12× 区间
4. **Earnings Power Value EPV** — 假设无增长的盈利能力估值
5. **Yield-based valuation** — 给定要求收益率，反推合理价

关键假设：
- 永续增长率为负（- 1% ~ - 3%）
- WACC 8-10%
- 分红可持续性最关键
- 价值陷阱检测：注意"低 PE"不等于"便宜"

## 七、同业对比策略

- **同业范围**：同行业（电信 = AT&T vs Verizon vs T-Mobile；烟草 = Altria vs PMI vs BAT；传统消费品 = KHC vs GIS vs CPB）
- **同业数量**：3-5 家
- **对比维度**：分红可持续性、FCF 稳定性、转型进展、衰退率

## 八、关键风险与黑天鹅

1. 衰退加速（如电子烟对烟草、流媒体对有线电视）
2. 监管打压（FDA 禁烟、欧盟限烟、美国 IRA 对药价压制）
3. 分红削减（最大单日跌幅触发器）
4. 评级下调
5. 业务转型失败（IBM 云、Kraft Heinz 高端化）
6. 主要客户 / 渠道流失
7. 不理性收购转型（Yahoo / AOL 案例）
8. 持续低估值 → 私有化 / 杠杆收购（PE 介入风险）

## 九、required_pct 默认值

`base = 40%`（V2_SPEC §六，反映衰退不确定性 + 价值陷阱风险）

调整：
- 高质量 cash cow（Altria、中移动）：40% - 5% = **35%**
- 衰退加速 / 转型挣扎：+10 = **50%**
- 含价值陷阱信号：+10

## 十、数据源补充

- 主要数据源 1：年报 + 季报（FCF / 派息 / Capex）
- 主要数据源 2：行业用户数据（移动 ARPU / 用户数趋势、烟草销量）
- 主要数据源 3：分红连续性（Dividend Aristocrat / Champion 名单）
- 主要数据源 4：监管动态（FDA 烟草、FCC 电信）
- 主要数据源 5：评级机构变更通知

## 十一、典型案例参考

**Altria MO（万宝路）**：传统烟草巨头，烟民人数下降但通过涨价维持 FCF 稳定。分红率 ≥ 80%，股息率 8%+，连续 50+ 年分红。投资 JUUL 电子烟（亏损 $100 亿+）和 Cronos 大麻均失败。面临 FDA 薄荷烟禁售、电子烟监管反复、欧盟限烟等。required_pct ≈ 35-40%。叠加 `[regulated_heavily]`。

**中国移动 0941.HK / 600941.SH**：4G / 5G 网络龙头，用户 9 亿+ 但 ARPU 增长有限。2021 年回 A 后股息率 7%+，分红率 70%+。在云 / 数字化业务上转型尝试。叠加 `[a_h_dual_listed, state_owned, regulated_heavily]`。
