# Sector: 困境反转 (`distressed_turnaround`)

**典型公司**：
- A 股：福耀玻璃（2008 早期反转案例）、伊利股份（三聚氰胺后）、海康威视（被制裁压制期）
- 港股：太古地产（2020 疫情）、各种 spinoff
- 美股：Boeing BA、Bed Bath & Beyond（已破产）、GameStop（部分时段）、Wells Fargo（2016 假账后）、AIG（2008 后）

**v2 schema_extension**：无（base schema；可酌情按主行业另叠 distressed modifier）
**主流派路由**：霍华德·马克斯（困境投资专家） / 巴菲特（American Express 早年）

**默认 modifier_flags**：必带 `distressed`

## 一、公司原型识别信号

- 历史上是优质或中等以上公司，因短期事件大幅下跌（≥ 50% from peak）
- 主业未根本失效，但财务 / 治理 / 战略受重创
- 股价跌至 NAV / Book Value 以下 或 PE / EV/EBITDA 大幅低于同业
- 市场情绪极度负面（卖方目标价 vs 内在价值差距大）
- 内部已采取 turnaround 措施（更换 CEO、剥离亏损业务、削减成本、再融资）
- 资产质量仍有价值（品牌 / 技术 / 牌照 / 客户 / 现金流潜力）

## 二、商业模式六维适配

- **价值主张**：原有价值主张仍存在但受冲击
- **收入结构**：通常下滑，关注修复路径
- **成本结构**：可能臃肿，turnaround 重点
- **客户结构**：可能流失部分客户但留有核心
- **再投资循环**：资本开支 / 营运资金 / 偿债压力大
- **关键资源**：取决于行业，重点考察"是否还在"

## 三、护城河重点

最适用：**取决于主业** —— 关键问题是"护城河是受暂时冲击还是永久受损"

锚点（与原始行业相比）：
- 7 分：护城河仍在但被短期事件压制（Boeing 飞机制造寡头壁垒仍在）
- 5 分：护城河受冲击但可修复（Wells Fargo 假账后客户信任修复）
- 3 分：护城河可能永久受损（部分零售商被亚马逊冲击）
- 0 分：实质破产 / 价值归零

验证：
- 是 cyclical 困境（油价 / 经济衰退） vs structural 困境（被颠覆）？
- 管理层是否已识别问题并行动？
- 资产是否仍有重置价值？
- 流动性是否足够撑过 turnaround 期（≥ 24 个月 runway）？

## 四、核心财务指标

| 指标 | 优秀 / 信号 | 警戒 / 死亡螺旋 |
|---|---|---|
| 流动性（现金 + 信贷额度 / 18 月烧钱） | ≥ 1.5×（撑得过） | ≤ 0.5×（撑不过） |
| Net Debt / EBITDA（Normalized） | ≤ 4× | ≥ 6× |
| 利息覆盖倍数 | ≥ 2× | ≤ 1× |
| FCF 转正时间预期 | < 24 月 | > 36 月或不明确 |
| 营收稳定性（季度环比） | 触底回升 | 持续下滑 |
| 毛利率（季度趋势） | 触底回升 | 持续恶化 |
| 客户留存 / 订单情况 | 核心客户未流失 | 主要客户流失（Boeing 航司转 Airbus） |
| 评级 | 高收益级但稳定 | 临近违约（CCC） |
| 资产可剥离价值（非核心资产） | 显著 + 已开始处置 | 无变现路径 |
| 管理层换血 + 战略清晰度 | 新 CEO 已上任 + 战略书披露 | 不变 / 混乱 |
| 监管 / 诉讼负债已计提 | 充分计提 | 仍在扩大 |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1-F7 多数财务红旗本身就是触发 distressed 的原因，但仍要逐项过审
- **改读项**：
  - 已发生的红旗本身不是新增风险，而是已知风险
  - 关键看"风险是否在收敛"还是"仍在扩大"
- **新增类型专属红旗（distressed 特有）**：
  - DT1：流动性 runway < 12 月（生死红线）
  - DT2：评级被下调至 CCC 或 D
  - DT3：核心客户流失（Boeing 大额订单取消）
  - DT4：监管 / 诉讼负债持续扩大（特别是未来损失计提不足）
  - DT5：管理层 turnaround 计划已宣告失败（如重组进展不及预期）
  - DT6：再融资失败 / 债权人拒绝展期
  - DT7：核心资产 / 品牌价值开始永久受损
  - DT8：现有大股东 / 控股股东撤退

## 六、估值方法优先级

1. **资产基础 / 重置成本 / 清算价值** — 安全底估算
2. **SOTP** — 各分部独立估值（可处置部分单独估值）
3. **Normalized PE / EV/EBITDA** — 用 turnaround 完成后 EPS / EBITDA 估值
4. **DCF（三情景 + 概率加权）**:
   - Bull: 完全 turnaround，回到原始估值
   - Base: 部分 turnaround
   - Bear: turnaround 失败，资产清算
5. **Reverse DCF** — 检验当前市值隐含什么情景

关键假设：
- 概率加权各情景（如 30% bull / 50% base / 20% bear）
- 折现率 WACC 12-15%（困境风险溢价）
- 必须有"撑得过"分析（runway 测算）

## 七、同业对比策略

- **同业范围**：与原行业同业 + 历史困境反转案例（AIG / Wells Fargo / Tyco）
- **同业数量**：3-5 家（含历史案例）
- **对比维度**：流动性、负债结构、主业修复路径、管理层质量

## 八、关键风险与黑天鹅

1. Turnaround 失败导致破产（Bed Bath & Beyond 案例）
2. 评级下调连锁触发再融资失败
3. 监管 / 诉讼负债持续扩大（Boeing 737 MAX 系列事故 + 后续 Door Plug 事件）
4. 核心客户 / 订单流失加速
5. 管理层频繁更换 / 战略漂移
6. 行业结构性恶化（不是周期性困境）
7. 大股东撤退引发市场恐慌
8. 价值陷阱（公司"看起来便宜"但持续恶化）—— 这是 distressed 最大风险

## 九、required_pct 默认值

`base = 50%`（V2_SPEC §六，最高之一）

调整：
- 优秀困境标的（明确 turnaround + 充足流动性）：50% - 0（quality 仍偏低）+ 10（distressed）= **60%**
- 真正深度困境：**70%+** 仍需保守
- 含 PCAOB / VIE：再加

## 十、数据源补充

- 主要数据源 1：年报 + 季报 + 8-K 重大事件公告
- 主要数据源 2：评级机构（穆迪 / 标普 watch list）
- 主要数据源 3：CDS 价格 / 高收益债二级市场（隐含违约概率）
- 主要数据源 4：管理层 turnaround 计划披露
- 主要数据源 5：行业新闻 / 监管诉讼追踪

## 十一、典型案例参考

**Boeing BA**：2018-2019 737 MAX 两起空难 → 全球停飞 + 重大监管罚款；2024 Alaska Airlines Door Plug 事件 + 罢工 + 质控危机；负债大幅扩张（净负债 500+ 亿美元）+ FCF 转负。生死关键：航司订单 backlog（仍 5000+ 架）+ 双寡头壁垒（与空客）+ 国防业务现金流。Turnaround 时间 36-48 月，需观察 Kelly Ortberg 新 CEO 战略执行。required_pct ≈ 60%+。叠加 `[distressed]`。

**Wells Fargo WFC**：2016 假账丑闻后股价大跌 + 监管罚款 + Fed 资产规模上限。2020 Charlie Scharf 上任 CEO + 战略瘦身 + 合规建设。2024 资产上限逐步解除，ROE 修复至 12%+。可作 distressed_turnaround 成功案例。
