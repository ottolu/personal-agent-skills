# Sector: 伟大公司复利机器 (`great_company_compounder`)

**典型公司**：
- A 股：贵州茅台、海天味业（早期）、伊利股份（部分时段）
- 港股：腾讯控股（重叠 tech_platform_network，按主行业归 tech）
- 美股：可口可乐 KO、Moody's MCO、S&P Global SPGI、Visa V、万事达 MA、Costco COST

**v2 schema_extension**：无（使用 base schema）
**主流派路由**：巴菲特 / 段永平（"难得糊涂"哲学）

## 一、公司原型识别信号

满足以下 4-6 条 →归入此类（高门槛）：

- 连续 10 年 ROIC ≥ 20%（含 GFC、疫情、行业周期）
- 连续 10 年 ROE ≥ 15% 且大部分年份 ≥ 20%
- 经营性现金流连续 10 年 > 净利润（OCF/NI ≥ 1.0）
- 营收 CAGR ≥ 8%（10 年），且不来自激进并购
- 自由现金流稳定，分红 + 回购占 FCF 比例 ≥ 50%
- 拥有多重护城河（品牌 + 网络效应 / 品牌 + 规模 / 标准化协议 + 切换成本 等至少 2 项）

## 二、商业模式六维适配

- **价值主张**：解决高频 / 必需 / 难以替代的需求（饮料、调味、信用评级、支付清算）
- **收入结构**：现金循环极佳（消费品预付款、Visa 按笔抽佣）；定价权随通胀传导
- **成本结构**：高毛利率（≥ 50% 常见，茅台 90%+）；轻资产或资产高效（COGS 占比低）
- **客户结构**：极度分散（终端消费者 / 千万商户），无单一客户依赖；客户复购率高
- **再投资循环**：再投资率低（capex/OCF < 30%），多余现金返还股东
- **关键资源**：品牌（无形资产）、专利 / 标准、规模经济、网络效应

## 三、护城河重点

最适用：**品牌护城河 + 规模 + 网络效应组合**

护城河 4 分锚点：
- 10 分（多重壁垒持续变宽）：茅台（品牌 + 稀缺产能 + 渠道）、Moody's（标准 + 双寡头 + 监管壁垒）
- 7 分（宽护城河 / 多来源）：可口可乐（品牌 + 全球分销）
- 验证方法：(a) 毛利率/净利率长期稳定或扩张；(b) 涨价后销量未下滑；(c) 新进入者尝试均失败

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| ROIC（10 年中位） | ≥ 20% | < 15% |
| ROE（10 年中位） | ≥ 20% | < 15% |
| 毛利率 | ≥ 50%（消费品） / ≥ 70%（数据型） | 持续下降 |
| OCF / Net Income (10 年中位) | ≥ 1.0 | < 0.8 |
| FCF / Net Income | ≥ 0.8 | < 0.6 |
| Capex / OCF | ≤ 30% | > 50% |
| 净现金 / 总资产 | 净现金 ≥ 0 | 净负债率 > 30% |
| 分红 + 回购 / FCF | ≥ 50% | < 30%（无再投资机会却囤现金） |
| 营收 CAGR（10 年） | ≥ 8% | < 5% |
| EPS CAGR（10 年） | ≥ 10% | < 5% |

## 五、红旗清单 N/A / 改读规则

针对 v1 32 项红旗：
- **N/A 项**：F10（杠杆经营依赖）— 此类型本身现金堡垒；F26（频繁会计准则变更）— 通常会计极简
- **改读项**：
  - F8（应收账款增速 > 营收增速）：消费品预收账款应反向考察（茅台预收款下滑反而是红旗）
  - F14（SBC 占比高）：N/A 适用范围；若是软件型平台则按 tech_platform 改读
- **新增类型专属红旗**：
  - GR1：连续 2 年 ROIC 跌出 20% 区间 → 护城河松动预警
  - GR2：管理层突然大规模并购非核心业务 → 资本配置质量警报
  - GR3：分红 + 回购停止但无大资本开支说明 → 现金被错误留存
  - GR4：核心产品涨价后销量下滑 → 定价权预警

## 六、估值方法优先级

按 V2_SPEC 估值工具箱：

1. **DCF（10 年明确预测期 + Gordon 永续）** — 主估值方法，永续增长率 2-3%（不超过名义 GDP）
2. **Earnings Power Value (EPV)** — 假设无增长的盈利能力底价
3. **Reverse DCF** — 从当前价反推市场隐含增长率，与基本面对比
4. **历史 PE / PB 估值带 + 相对估值** — 辅助参考，找 -1σ / -2σ 区间

关键假设：
- 永续增长率 ≤ 3%
- WACC 取 8-10%（中国 9-10%，美国 8-9%）
- 不要给"伟大"以过高永续增长率溢价，宁可保守

## 七、同业对比策略

- **同业范围**：跨市场（茅台 vs 帝亚吉欧；可口可乐 vs 雀巢；Moody's vs S&P Global）
- **同业数量**：3-5 家全球龙头
- **对比维度**：ROIC、毛利率、品牌溢价、海外收入占比、分红率

## 八、关键风险与黑天鹅

1. 消费者偏好长期迁移（如健康意识抑制含糖饮料）
2. 监管反垄断（Visa/Master 在欧美的费率压制）
3. 替代品技术变革（数字支付对现钞、AI 对评级业务的冲击）
4. 创始人 / 关键人物突变（茅台前董事长贪腐被查、可口可乐董事会变动）
5. 估值过高（最大风险——好公司 + 烂价格 = 烂回报，2000 年 PG 教训）
6. 货币 / 关税摩擦（涉及全球分销的品牌）
7. 公司过度回报股东导致再投资不足，错过新平台机会
8. 内部腐败 / 渠道压货（茅台经销商体系）

## 九、required_pct 默认值

`base = 25%`（V2_SPEC §六）

调整后典型范围：
- 高质量优秀公司：25% - 5% = **20%**
- 普通伟大公司：**25%**
- 估值合理但短期承压：25% + 0 = **25%**

## 十、数据源补充

- 主要数据源 1：年报 10-K / 中报（关注 segment 数据 + capital allocation 章节）
- 主要数据源 2：致股东信（特别是有 CEO 个人风格的公司，如可口可乐、Berkshire 旗下）
- 主要数据源 3：行业品牌价值榜（Interbrand / Brand Finance）
- 主要数据源 4：Wide Moat ETF (MOAT) 成分股变化追踪

## 十一、典型案例参考

**贵州茅台 600519.SH**：连续 ROIC > 30%、品牌 + 稀缺产能 + 渠道三重护城河、近 100% 毛利率，预收账款反映需求强度。风险：政商关系、年轻人偏好变化、海外渗透低。

**可口可乐 KO**：巴菲特持仓近 40 年，全球分销 + 浓缩液模式 + 品牌组合。1998 年 PE 高达 50× 后停滞 14 年，证明"伟大 + 高估 = 长期低回报"。
