# Sector: 深度低估烟蒂股 (`cigarbutt_deep_value`)

**典型公司**：
- A 股：（少见，A 股估值底相对偏高）部分小盘破净股
- 港股：（常见）大量小盘股长期 PB < 0.5 / 净净股（NCAV positive）
- 美股：（巴菲特早年）See's Candy 之前的 Dempster Mill、Sanborn Map、Berkshire Hathaway（原始纺织厂）；日本经济泡沫后大量"净净股"
- 日本：低 PB（< 1）+ 大量现金占市值的中小盘股（巴菲特 2020 买入五大商社部分案例）

**v2 schema_extension**：无（base schema）
**主流派路由**：格雷厄姆（净净股 NCAV 框架）— v2 中唯一以格雷厄姆为主流派的 sector

## 一、公司原型识别信号

- **NCAV Positive**: 流动资产 - 总负债 > 0 且 ≥ 市值 × 1.5（格雷厄姆经典定义）
- **PB < 0.7**（极度便宜，常见 < 0.5）
- **PE < 5**（如有正盈利）或当期亏损但累计盈利正
- **市值 < 净现金**（罕见，更便宜情况：净现金 + 投资性证券 > 市值）
- 通常是平庸或下滑业务，但有资产基础保护
- 多为小盘股 / 流动性差 / 卖方关注少
- 资产质量需逐项审查（应收账款实质性 / 库存可变现性 / 商誉减值已充分）

## 二、商业模式六维适配

- **价值主张**：业务通常平凡 / 同质化 / 下滑中
- **收入结构**：稳定但增长有限或负
- **成本结构**：通常偏臃肿（典型烟蒂特征）
- **客户结构**：客户基础萎缩或老化
- **再投资循环**：通常已停止扩张，现金积累或还债
- **关键资源**：账上现金 / 短期投资 / 应收 / 存货 / 不动产 / 上市公司控股权

## 三、护城河重点

最适用：**资产保护（清算价值 / 净流动资产）** —— 不依赖经营护城河

锚点：
- 7 分：（罕见）净净股 + 业务尚有现金流（短暂的"格雷厄姆雪茄"）
- 5 分：清算价值充足但运营恶化
- 3 分：清算价值临界 + 运营亏损（接近价值陷阱）
- 0 分：资产质量造假 / 应收商誉减值未充分（伪烟蒂）

注意：格雷厄姆经典策略要求**分散持仓 30+ 只**烟蒂股（单只胜率 < 50%，但组合期望正），不适合集中持仓。

## 四、核心财务指标

| 指标 | 烟蒂候选 / 优秀 | 价值陷阱 / 警戒 |
|---|---|---|
| NCAV / 市值 | ≥ 1.5×（深度烟蒂） | < 1× |
| Net Cash / 市值 | ≥ 0.5×（罕见）| < 0.2× |
| PB | < 0.5（深度低估） | > 1.0 |
| PE TTM | < 5（有盈利） | 持续亏损扩大 |
| 应收账款 / 现金 | < 0.5×（不依赖应收变现） | > 1.5× |
| 库存周转 | 合理（180-365 天） | > 730 天（积压严重）|
| 商誉 / 净资产 | < 10%（资产实在）| > 30%（含水分） |
| FCF 历史趋势 | 至少不严重恶化 | 加速恶化（烧钱） |
| 净负债 / NCAV | < 30%（杠杆可控） | > 80% |
| 实际控制人持股 | < 50%（不至于完全锁死） | > 75%（流动性 + 治理风险） |
| 分红 / 回购意愿 | 有分红 / 偶有回购 | 完全无回报股东行为 |
| 触发清算 / 私有化的概率 | 有迹象（大股东增持、要约） | 无任何迹象 |

## 五、红旗清单 N/A / 改读规则

格雷厄姆框架下，多数红旗本身就是"已知风险"，不构成新增警戒。但必须严查资产质量：

- **N/A 项**：F1（营收增速）— 烟蒂股不期待增长；F7（ROIC 低）— 默认低 ROIC
- **改读项**：
  - F2（应收账款 > 营收）：**关键** —— 应收账款是否实质可收回？是否大额减值未计提？
  - F22（存货周转过慢）：**关键** —— 存货实质可变现性？跌价准备充分？
  - F23（商誉占比高）：**关键** —— 商誉是否还应减值？（许多烟蒂股商誉是"幽灵资产"）
  - F26（频繁会计准则变更）：警惕"账面美化"
- **新增类型专属红旗**：
  - CB1：应收账款 / 存货大幅减值未充分计提（资产负债表水分）
  - CB2：商誉占净资产 > 30%（PB < 1 但商誉减后实际更高）
  - CB3：大股东 / 实控人长期减持（内部人看坏）
  - CB4：审计师变更频繁（披露质量风险）
  - CB5：净流动资产中含大量长期投资 / 限制使用资金（非真正流动）
  - CB6：连续多年现金流为负 + 还在烧（很快 NCAV 也会消失）
  - CB7：业务结构性消失（典型价值陷阱 —— 烟蒂被吸完）
  - CB8：港股 / 小盘流动性极低（无法退出）

## 六、估值方法优先级

1. **NCAV 净流动资产价值法** — 格雷厄姆经典：流动资产 - 总负债（不计算固定资产）
2. **清算价值法** — 各类资产按可变现折扣率（应收 80%、存货 50%、PP&E 30-50%）
3. **PB + 资产质量调整** — PB < 0.7 后逐项资产打分
4. **Sum-of-Parts** — 多业务 / 多分部分别估值
5. **Reverse 思维**：当前价格隐含什么"被吸走"程度

关键假设：
- 不依赖未来盈利（DCF 不适用）
- 资产质量打分是核心
- 折现率 / WACC 不重要（资产价值 vs 现值）
- 组合分散是基本前提

## 七、同业对比策略

- **同业范围**：同行业相比（很多烟蒂集中在传统制造 / 香港小盘 / 日本中盘）
- **同业数量**：3-5 家烟蒂股池
- **对比维度**：NCAV / 市值、净现金、资产质量、控股股东行为、流动性

## 八、关键风险与黑天鹅

1. **价值陷阱（核心风险）** —— 业务持续消失，NCAV 也被吸走（伯克希尔纺织厂教训）
2. 资产质量造假 / 应收商誉水分未计提
3. 大股东低价私有化 / 私下转让（小股东被损害）
4. 控股股东价值输送 / 关联交易
5. 治理结构问题 / 流动性陷阱
6. 长期不分红 / 不回购，现金被持续"看管"
7. 港股 / 日股小盘流动性差，无法退出
8. 会计准则变更掩盖资产减值

## 九、required_pct 默认值

`base = 50%`（V2_SPEC §六，最高之一）

调整：
- 真正深度烟蒂（NCAV/市值 ≥ 2×）：50% - 5%（资产保护够强） = **45%**
- 普通烟蒂：**50%**
- 含 distressed + 价值陷阱信号：+10 = **60%**

注意：格雷厄姆方法要求 NCAV ≥ 1.5× 市值 才购买（自带 33% 安全边际），加上 required_pct 后等于"双重保险"

## 十、数据源补充

- 主要数据源 1：年报 + 季报（资产负债表逐项审查）
- 主要数据源 2：内部人持股变动（实控人 + 高管增持 / 减持）
- 主要数据源 3：股东大会决议（分红 + 回购 + 私有化议程）
- 主要数据源 4：审计师报告（关注审计意见类型）
- 主要数据源 5：香港 / 日本 / 港股小盘行情数据（流动性评估）
- 主要数据源 6：格雷厄姆体系网站（如 NCAV screener）

## 十一、典型案例参考

**伯克希尔哈撒韦原始纺织厂（1962-1985）**：巴菲特最著名的"烟蒂教训"。当年 PB ≈ 0.5，看似深度低估，但纺织业被合成纤维和海外低成本生产持续吸蚀，巴菲特最终被迫关闭原始业务。后转型为投资控股平台。教训："以合理价格买入伟大公司远胜以便宜价格买入平庸公司。"

**日本五大商社（巴菲特 2020 - 至今）**：伊藤忠、丸红、三井物产、住友商事、三菱商事。2020 买入时 PB < 0.7，PE < 7，股息率 5%+，净现金占市值 ≥ 20%。叠加日本市场制度性折价。2024 巴菲特持仓持续增至 9.9%。虽非纯净净股，但近似深度低估 + 资产保护逻辑。可叠 `[adr_listed]`（部分以 ADR 上市）。
