# Sector: 公用事业/基础设施/REIT (`utility_infrastructure_reit`)

**典型公司**：
- A 股：长江电力 600900.SH、国电南瑞、华能水电
- 港股：领展房产基金 0823.HK、长江基建 1038.HK、电能实业 0006.HK
- 美股：NextEra Energy NEE、Duke Energy DUK、Realty Income O、American Tower AMT、National Grid NGG（伦敦）

**v2 schema_extension**：`utility_infrastructure_reit.json`（DDM 参数 / regulated_return / 派息率历史 / 资产基础）
**主流派路由**：巴菲特（旗下 BHE Berkshire Hathaway Energy） / 股息派 / 防御性配置

**默认 modifier_flags**：通常带 `regulated_heavily`；中资可叠 `state_owned`

## 一、公司原型识别信号

- 主营受规管业务（电网 / 自来水 / 燃气 / 公路 / 通信塔 / REIT 收租）
- 资本基础（Rate Base）回报率受监管核定（如美国 ROE 通常 9-10%）
- 经营性现金流极度稳定，可预测性高
- 高分红率（≥ 60% Payout Ratio 常见，REIT 法定 90%+）
- 通常资本密集 + 高杠杆（负债 / EBITDA 5-8×）
- 长期合同 / 特许经营权 / 收费权

## 二、商业模式六维适配

- **价值主张**：稳定提供必需公共服务 / 基础设施 / 收租
- **收入结构**：受规管费率 × 用量 / 长期 PPA 合同 / 租金
- **成本结构**：固定折旧 + 维护 + 利息；运营杠杆高
- **客户结构**：居民 / 工商业 / 政府；多元分散
- **再投资循环**：扩大资本基础以获取更多受规管回报；新能源投资
- **关键资源**：特许经营权 / 资产基础 / 监管关系 / 资本能力

## 三、护城河重点

最适用：**特许经营权 + 监管壁垒 + 自然垄断**

锚点：
- 10 分：（罕见）长江电力（三峡 + 葛洲坝 + 溪洛渡 + 向家坝独家水电）
- 7 分：NEE（电力 + 新能源行业最强）、Realty Income（NNN REIT 模式 + 50 年分红史）、领展（香港优质零售物业）
- 5 分：普通 REIT / 公用事业（受规管回报）
- 验证：(a) 无替代供应商；(b) 长期合同 / 特许期 ≥ 20 年；(c) 监管回报率 ≥ 资本成本

## 四、核心财务指标（参考 schema_extension: utility_infrastructure_reit.json）

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 营收增长率 | 2-5%（公用事业可接受低速） | 持续负增长 |
| EBITDA Margin | ≥ 40%（公用）/ ≥ 65%（REIT） | < 30% / < 50% |
| 自由现金流（覆盖股息）| FCF Payout < 90% | FCF Payout > 100%（举债分红） |
| Net Debt / EBITDA | ≤ 6× | ≥ 7×（再融资压力） |
| 利息覆盖倍数 | ≥ 3× | ≤ 2× |
| 受规管资产占比 | ≥ 80%（受规管资产保护） | ≤ 50%（贝塔过强） |
| ROE | 8-12%（受规管核定） | < 5% 或负 |
| Payout Ratio | 65-85% | > 100%（不可持续） |
| AFFO / NOI 增长（REIT） | ≥ 5% | < 0% |
| Same-Store NOI 增长（REIT） | ≥ 3% | < 0% |
| Funds From Operations 增长 | ≥ 5% | < 0% |
| 资本基础 Rate Base 增速 | ≥ 5% | < 0% |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1（营收增速）— 低速增长是常态；F7（ROIC 低）— 受规管行业 ROIC 天花板就低；F22（存货周转）— N/A
- **改读项**：
  - F10（高杠杆）：行业本身高杠杆（5-8× EBITDA），需与同业比较，不是绝对红旗
  - F12（资产负债率）：N/A
  - F19（现金枯竭）：如 FCF < 股息支出则警报
- **新增类型专属红旗**：
  - UT1：监管下调允许回报率（如美国 ROE 从 10% 调到 9%）
  - UT2：FCF 不足以覆盖股息（典型 REIT 举债分红风险）
  - UT3：评级下调（投资级 → 高收益）触发再融资困难
  - UT4：核心特许经营权到期且续约不确定
  - UT5：REIT 主要租户大额违约（如 Realty Income 早年 GE 子公司违约）
  - UT6：利率快速上行冲击 REIT 估值（2022 - 2024 全球 REIT 大跌）
  - UT7：环保事故 / 安全事故（如 PG&E 加州山火）

## 六、估值方法优先级

1. **DDM 股息折现模型** — 主方法（高分红、稳定增长）
2. **AFFO / FFO 倍数法** — REIT 标准估值方法
3. **NAV 净资产价值** — REIT 持有物业的资产估值
4. **DCF（受规管回报建模）** — 适用受规管公用事业
5. **RAB × WACC** — 监管资产基础 × 受规管回报率
6. **EV/EBITDA** — 跨同业可比

关键假设：
- 永续增长率 2-3%（公用事业）/ 3-4%（高质 REIT）
- 折现率 Ke 取 7-9%（防御性资产）
- 利率敏感性测试必做（每 +100bps 估值 -10~-15%）

## 七、同业对比策略

- **同业范围**：分类（电力 = 长江电力 vs Duke vs NEE；REIT 子类 = NNN / Office / Mall / Logistics 各自比较）
- **同业数量**：3-5 家同子类
- **对比维度**：派息率、AFFO 增长、Net Debt/EBITDA、出租率、SSNOI 增长

## 八、关键风险与黑天鹅

1. 利率快速上行（公用事业 + REIT 估值受压最严重）
2. 监管下调允许回报率
3. FCF 不足 / 举债分红难以为继
4. 核心特许经营权到期未续
5. 评级下调 / 信用利差扩大
6. 重大安全事故（PG&E 加州山火、福岛事故型）
7. 能源转型（传统火电 / 燃气资产搁浅 stranded asset）
8. REIT 主要租户违约（电商冲击零售 REIT）

## 九、required_pct 默认值

`base = 20%`（V2_SPEC §六，最低之一，反映稳定性）

调整：
- 长江电力级稀缺资产：20% - 5% + 3 = **18%**
- 普通公用 / REIT：20% + 0 + 3 = **23%**
- 利率上行周期：+5 ~ +10

## 十、数据源补充

- 主要数据源 1：年报 + 季报（FFO / AFFO / 资本基础 / 派息）
- 主要数据源 2：监管申请（FERC / 国家能源局费率核准）
- 主要数据源 3：REIT 持有物业明细（NAV 推算）
- 主要数据源 4：穆迪 / 标普评级报告
- 主要数据源 5：利率 / 国债收益率（贴现率敏感）

## 十一、典型案例参考

**长江电力 600900.SH**：拥有三峡、葛洲坝、溪洛渡、向家坝、白鹤滩、乌东德六座大型水电站，年发电量 ≈ 中国总发电量 3%。零燃料成本、零碳、特许经营、稀缺自然资源。ROE 15%+，分红率 ≥ 70%。叠加 `[state_owned, regulated_heavily]`。great_company 候选（已经从 utility 跃升）。

**Realty Income O**：美国最大 NNN REIT，2024 月度分红连续 55+ 年，630+ 次月度连续增加。主要租户包括沃尔玛、Dollar General、便利店等。可作 utility / REIT 子类的 great_company 候选。
