# Sector: 控股集团 (`holding_conglomerate`)

**典型公司**：
- A 股：复星医药（部分控股集团特征）、复星国际（港股 0656）
- 港股：长江和记 0001.HK、复星国际 0656.HK、太古股份 0019.HK
- 美股：Berkshire Hathaway BRK.A/B、Markel MKL、Loews L、Fairfax Financial FFH (Toronto)、Brookfield BN

**v2 schema_extension**：`holding_conglomerate.json`（segment_data / sotp_valuation / float_value / investment_portfolio_nav / 控股折价）
**主流派路由**：巴菲特（BRK 模式） / 段永平（控股投资视角）

## 一、公司原型识别信号

- 主营持有多个子公司股权（部分 100% 控股，部分少数股权）
- 经营性业务跨多个不相关行业（金融 + 工业 + 消费 + 投资组合）
- 资产负债表大量"长期股权投资"+"可供出售金融资产"
- 母公司层面通常无主营业务，靠子公司分红 + 投资组合
- 历史上有持续的资本配置案例（收购 / 处置 / 再投资）
- 通常存在"控股折价"（市值 < SOTP NAV 10-30%）

## 二、商业模式六维适配

- **价值主张**：CEO / 总部作为资本配置者，将子公司现金流和保险浮存金再投资到高回报机会
- **收入结构**：子公司合并报表 + 投资组合分红 / 利息 + 资本利得
- **成本结构**：总部费用通常 < 1% 总资产；子公司各自经营成本
- **客户结构**：N/A（取决于子公司）
- **再投资循环**：浮存金 + 子公司分红 + 资本利得 → 全权配置（核心 alpha）
- **关键资源**：CEO 资本配置能力 / 浮存金 / 子公司质量 / 投资组合

## 三、护城河重点

最适用：**CEO 资本配置能力（最特殊的护城河）+ 浮存金成本 + 集团协同效应**

锚点：
- 10 分：BRK（巴菲特 / 芒格 60 年复利 + 浮存金成本接近 0 + 子公司分散 + 现金弹药千亿）
- 7 分：Markel、Fairfax（小型 BRK 模式）、Brookfield（另类资产配置）
- 5 分：太古、长江和记（多元化但 CEO 资本配置一般）
- 3 分：（部分中资集团）复杂关联交易 + 价值毁灭

验证：
- 多年来 NAV 复合增长率（BRK 历史 ≥ 19%）
- CEO 历史资本配置回报（收购 IRR、回购时机）
- 子公司质量 + 多元化程度

## 四、核心财务指标（参考 schema_extension: holding_conglomerate.json）

| 指标 | 优秀 | 警戒 |
|---|---|---|
| Book Value Per Share 增速（5 年 CAGR） | ≥ 15% | ≤ 8% |
| SOTP NAV / 市值（折价） | 折价 20-30%（建仓区） | 溢价 > 10%（已透支） |
| 投资组合公允价值（季度披露） | 透明披露 | 不披露 / 滞后披露 |
| 浮存金规模与成本 | 大且接近 0 成本 | 成本 ≥ 5% |
| 子公司 ROIC 加权平均 | ≥ 15% | ≤ 10% |
| 总部费用 / 总资产 | < 1% | > 2% |
| 财务杠杆（净负债 / NAV） | ≤ 20% | ≥ 40% |
| 现金及流动资产 / NAV | 10-20%（有弹药） | < 5% 或 > 50%（错过机会） |
| 历史回购占 FCF 比 | 20-50%（机会主义） | < 5%（高估时仍未回购） |
| 子公司多元化（无单一 > 30%） | ≤ 30%（健康分散） | ≥ 50%（单点风险） |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1（营收增速）— 合并报表受子公司影响波动；F3-F5（毛利率）— N/A（多元化无统一毛利率）；F22（存货周转）— N/A；F23（商誉占比高）— 控股集团商誉天然高
- **改读项**：
  - F7（ROIC 低）：用 NAV 增速 + 加权子公司 ROIC 替代
  - F10（杠杆）：用净负债 / NAV 替代
  - F11（关联交易）：控股集团关联交易普遍，但需关注是否存在价值输送
- **新增类型专属红旗**：
  - HC1：CEO 资本配置质量持续下滑（连续多年大额减值）
  - HC2：子公司多元化失败（"多元化恶化器 diworsification"）
  - HC3：投资组合大幅减持 + 现金未有效再投资
  - HC4：复杂的关联交易 / 价值输送（部分中资集团案例）
  - HC5：核心 CEO 风险（巴菲特接班 / 个人健康）
  - HC6：核心子公司业务大幅恶化（保险浮存金成本飙升）
  - HC7：控股折价持续扩大至 > 40%（市场不信任）

## 六、估值方法优先级

1. **SOTP 分部估值（绝对主导方法）** — 各子公司 + 投资组合分别估值后加总，再扣总部负债
2. **NAV 净资产价值法** — 类似 SOTP 但更强调资产基础
3. **Look-Through Earnings（穿透盈利）** — 巴菲特方法：合并报表盈利 + 少数股权（持股 < 20%）应享盈利
4. **DCF**（子公司层面）— 各子公司单独 DCF 后加总
5. **历史 PB 估值带** — 辅助

关键假设：
- SOTP 中子公司估值取行业可比倍数
- 投资组合按市价 / 公允价值
- 控股折价：通常 10-30%（取决于流动性 / 透明度 / CEO 质量）
- 总部负债从 SOTP 价值中扣减

## 七、同业对比策略

- **同业范围**：全球（BRK vs Markel vs Fairfax vs Brookfield；亚洲 = 长江和记 vs 太古 vs Jardine Matheson）
- **同业数量**：3-5 家或宣告 incomparability_flag（BRK 独一档难比）
- **对比维度**：NAV 增速、控股折价、子公司质量、CEO 配置历史

## 八、关键风险与黑天鹅

1. CEO 接班风险（巴菲特 / 芒格 / Watsa / Pinault 个人风险）
2. 子公司业务大幅恶化（如 BRK 的 BNSF 铁路 / Geico）
3. 投资组合集中暴雷（如 BRK 单一持仓减值）
4. 浮存金成本飙升（如保险 COR > 100%）
5. 控股折价持续扩大（市场失去信心）
6. 多元化恶化（diworsification）
7. 关联交易 / 价值输送（治理瑕疵）
8. 监管 / 反垄断（持股超 10% 触发申报）

## 九、required_pct 默认值

`base = 20%`（V2_SPEC §六，反映稳定性 + CEO 资本配置 alpha）

调整：
- BRK / Markel 级：20% - 5% = **15%**
- 普通控股集团：**20%**
- CEO 资本配置历史平平 + 透明度低：+10 = **30%**
- 含 distressed：+10 = **30%+**

## 十、数据源补充

- 主要数据源 1：年报（每年的合并报表 + 子公司分部）

## 十一、典型案例参考

**Berkshire Hathaway BRK.A / BRK.B**：巴菲特 / 芒格 60 年复利典范。子公司涵盖保险（Geico / BHE）+ 铁路（BNSF）+ 工业（Marmon / Precision Castparts）+ 消费（See's Candies、Dairy Queen）+ 能源（BHE）+ 大量上市公司持股（苹果、可口可乐、美国银行、Coca-Cola 等）。浮存金 1500+ 亿美元。Look-Through Earnings 2024 ≈ 700 亿美元。控股折价历史 0-20%，部分时段溢价。

**复星国际 0656.HK**：中国版"巴菲特模式"尝试，跨医药、保险、地产、文旅、消费多元业务。2022 流动性危机后大规模资产处置（葡萄牙 BCP、Atlantia 等），控股折价一度 > 40%。可作为控股集团失败案例参考。
