# Sector: 地产 (`real_estate`)

**典型公司**：
- A 股：万科 A 000002.SZ、保利发展、招商蛇口
- 港股：万科企业 2202.HK、龙湖集团 0960.HK、华润置地 1109.HK、碧桂园 2007.HK
- 美股：D.R. Horton DHI、Lennar LEN、Toll Brothers TOL

**v2 schema_extension**：`real_estate.json`（NAV / 土储 / 净负债率 / 合约负债 / 销售金额 / 现金短债比 / 三道红线）
**主流派路由**：霍华德·马克斯（周期 + 困境投资） / 价值陷阱识别（典型案例）

**默认 modifier_flags**：A+H 上市常见 `a_h_dual_listed`；中资国企叠 `state_owned`；困境企业必带 `distressed`

## 一、公司原型识别信号

- 主营房地产开发销售（住宅 / 商业 / 工业地产）
- 营收高度周期性，与地产销售周期同步
- 资产负债表大量预收款（合同负债）+ 大额库存（土地 + 在建 + 已完工待售）
- 杠杆率高（净负债 / 净资产 ≥ 60%）
- 监管关注三道红线（中国）
- 长项目周期（拿地 → 开工 → 预售 → 交付 18-36 个月）

## 二、商业模式六维适配

- **价值主张**：满足居住需求 / 商业地产收租
- **收入结构**：销售（一次性）+ 物业管理（持续）+ 商业地产租金
- **成本结构**：土地（占比 30-50%）+ 建安 + 财务费用 + 销售管理
- **客户结构**：购房者（C 端）+ 商业租户（B 端）
- **再投资循环**：拿地 → 开发 → 销售回款 → 拿地循环；强周期、强政策敏感
- **关键资源**：土储 / 品牌 / 融资能力 / 城市选址

## 三、护城河重点

最适用：**品牌 + 城市卡位 + 融资能力 + 持有物业（自持商业 REIT 化）**

注意：开发型房企护城河相对脆弱（同质化 + 强周期 + 政策风险），自持商业型（华润、龙湖天街、领展）更接近 utility / REIT。

锚点：
- 7 分：龙湖、华润置地（多元业态 + 商业地产占比高 + 财务稳健）
- 5 分：万科（曾经龙头，2024 困境）
- 3 分：碧桂园（重仓三四线 + 高杠杆爆雷案例）
- 0 分：恒大（已破产）

## 四、核心财务指标（参考 schema_extension: real_estate.json）

| 指标 | 优秀 | 警戒（红色） |
|---|---|---|
| 净负债率（净有息负债 / 净资产） | ≤ 70%（三道红线绿档） | ≥ 100%（红档） |
| 剔除预收款的资产负债率 | ≤ 70% | ≥ 80% |
| 现金 / 短债（现金短债比） | ≥ 1.5× | ≤ 1× |
| 销售金额（年） | 持续增长 / 稳定 | 同比 -30% 以上（断崖） |
| 销售金额 / 总市值 | ≥ 1×（隐含 NAV 折价大） | < 0.3 |
| 合同负债（预收款）/ 营收 | ≥ 0.8× | < 0.3×（未来 1 年营收承压） |
| 已售未结面积 / 当年销售 | ≥ 1× | < 0.5× |
| 土地储备货值 / 年销售 | 3-5×（合理） | < 1.5×（无可售）或 > 8×（过度囤地） |
| 一二线土储占比 | ≥ 60% | < 30%（高线城市撤退） |
| 持有物业（自持）NOI 增长 | ≥ 5% | 下降 |
| 持有物业 / 总资产 | ≥ 30%（接近 REIT 业态） | < 10%（纯开发） |
| 经营性现金流 | 持续为正 | 持续为负（持续输血） |
| 平均融资成本 | ≤ 5%（中资） | ≥ 8%（高收益债评级） |
| 评级 | 投资级（BBB-/Baa3 以上） | 高收益（BB+/Ba1 以下） |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F22（存货周转过慢）— 地产存货 2-3 年正常
- **改读项**：
  - F1（营收增速 ≪ 行业）：行业整体下行，应改看合同销售额是否优于行业（< -30% 异常）
  - F4（毛利率）：行业毛利率近年从 30% 跌至 15%，需与同业对比
  - F8（应收账款 > 营收增速）：地产应收账款主要为已售未结，可关注
  - F10（杠杆）：必须用三道红线判定；融资成本高于行业为重红旗
  - F12（资产负债率 > 70%）：行业本身高负债，要用剔除预收款后口径
- **新增类型专属红旗**：
  - RE1：三道红线踩 ≥ 2 条（橙档 / 红档）
  - RE2：销售金额同比 -30% 以上连续 2 季度
  - RE3：现金短债比 < 1×（短期偿债危机）
  - RE4：境外债 / 美元债违约 / 展期（碧桂园、恒大、融创 案例）
  - RE5：评级下调至高收益级（CCC 以下意味重组）
  - RE6：拿地金额 / 销售金额 < 20% 持续（无可持续运转）
  - RE7：管理层动荡 / 控股股东变更（万科 2024 国资入主）
  - RE8：项目大规模停工 / 烂尾（保交楼困难）
  - RE9：商业地产空置率上升 / 租金下降

## 六、估值方法优先级

1. **NAV 净资产价值法** — 主方法。土储估值 + 在建工程 + 持有物业 + 现金 - 负债
2. **DCF（开发 + 持有物业分别建模）** — 严格区分一次性收入与持续 NOI
3. **PB 估值带 + 周期分位数** — 周期低位 PB 通常 0.3-0.6×，高位 1.5-2×
4. **SOTP** — 开发 + 商业 + 物管（部分港股已分拆物管子公司）
5. **股息折现** — 高分红地产（龙湖部分时段、华润置地）

关键假设：
- NAV 估算需对土储按城市能级 / 项目阶段分类折价（30-50% 折价是当前周期）
- 永续增长率 0-2%
- WACC 10-12%（困境期 + 政策风险）

## 七、同业对比策略

- **同业范围**：分类（中资国企 = 华润 vs 招商 vs 保利；中资民企 = 万科 vs 龙湖 vs 旭辉；美资 = DHI vs LEN）
- **同业数量**：3-5 家
- **对比维度**：三道红线、销售金额、毛利率、融资成本、持有物业占比

## 八、关键风险与黑天鹅

1. 销售断崖（限购 / 限贷 / 居民去杠杆）
2. 三道红线 / 融资收紧 / 美元债违约
3. 项目烂尾 / 保交楼 / 业主断供
4. 评级下调引发连锁再融资危机
5. 房价持续下跌侵蚀 NAV
6. 监管政策反复（"房住不炒" → 救市政策）
7. 国资入主稀释股东权益（万科 2024 案例）
8. 商业地产电商冲击 / 空置率上升
9. 信用证 / 商票违约（碧桂园模式）

## 九、required_pct 默认值

`base = 40%`（V2_SPEC §六，反映行业高不确定性）

调整：
- 优秀（华润、龙湖、DHI）：40% - 5% + 0 = **35%**
- 困境（万科 2024）：40% + 10（quality < 6.0）+ 10（distressed）+ 0（a_h_dual_listed）+ 0（state_owned）= **60%**
- 评级高收益级：+10 ~ +20

## 十、数据源补充

- 主要数据源 1：年报 + 月度销售公告（销售金额 / 面积）
- 主要数据源 2：克而瑞 / 中指院 行业销售排行
- 主要数据源 3：评级机构（穆迪 / 标普 / 中诚信）实时报告
- 主要数据源 4：美元债二级市场价格（隐含违约概率）
- 主要数据源 5：监管公告（三道红线、保交楼、城市更新政策）

## 十一、典型案例参考

**万科 A 000002.SZ / 2202.HK**：曾经的中国地产蓝筹龙头，"专业化、市场化、规模化"代表。2024 进入财务困境：销售金额腰斩、债务违约 / 展期、管理层动荡、深圳地铁国资入主成为实际控制人。V1 报告给出 PASS 4.30；V2 应仍 PASS，价值陷阱 6 信号识别更清晰（销售断崖 + 三道红线踩中 + 净负债率飙升 + 管理层动荡 + 评级下调 + 美元债违约展期），地产 schema_extension 必须使用，required_pct ≈ 60%。叠加 `[real_estate, distressed, a_h_dual_listed, state_owned]`。

**龙湖集团 0960.HK**：商业地产 + 物业服务 + 开发兼具的财务稳健民营房企。三道红线全绿档，平均融资成本 < 5%，天街自持商业组合 NOI 持续增长。在 2024 行业洗牌中相对稳健。
