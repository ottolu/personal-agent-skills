# Sector: 保险 (`insurance`)

**典型公司**：
- A 股：中国平安 601318.SH、中国人寿、新华保险、中国太保
- 港股：中国平安 2318.HK、友邦保险 1299.HK、保诚 2378.HK
- 美股：MetLife MET、Prudential PRU、Allstate ALL、Progressive PGR、Travelers TRV、Berkshire（兼控股集团）

**v2 schema_extension**：`insurance.json`（综合成本率 / 内含价值 EV / NBV 新业务价值 / 投资收益率 / 准备金 / 偿付能力）
**主流派路由**：巴菲特（保险浮存金核心策略）

**默认 modifier_flags**：必带 `regulated_heavily`；中资可叠 `state_owned`、`a_h_dual_listed`

## 一、公司原型识别信号

- 主营寿险 / 财险 / 健康险 / 再保险
- 拥有"浮存金"（Float）—— 已收保费但未赔付的资金，可用于投资
- 综合成本率 COR（财险）或新业务价值率 VONB（寿险）是核心指标
- 准备金占总负债比例极高（≥ 70%）
- 投资收益对净利润影响显著
- 监管核心指标：偿付能力充足率

## 二、商业模式六维适配

- **价值主张**：风险转移 + 财富积累 + 保障
- **收入结构**：保费收入 + 投资收益（双引擎）
- **成本结构**：赔付 + 营运 + 代理 / 渠道佣金 + 准备金计提
- **客户结构**：个人（寿险 / 健康险）+ 企业（财险 / 团险）
- **再投资循环**：浮存金投入固定收益 / 权益 / 另类资产
- **关键资源**：浮存金 / 精算能力 / 销售渠道 / 投资能力

## 三、护城河重点

最适用：**规模经济 + 监管壁垒 + 浮存金成本优势（巴菲特模型）**

锚点：
- 10 分：（罕见）Berkshire（浮存金成本接近 0 甚至负）、Progressive（科技驱动定价优势）
- 7 分：友邦（高端寿险 + 亚洲布局）、平安（生态闭环）
- 5 分：传统国有大型寿险（规模但效率一般）
- 验证：(a) 综合成本率 COR < 95%（财险）；(b) 新业务价值率 VONB > 30%（寿险）；(c) 投资收益率持续 > 行业

## 四、核心财务指标（参考 schema_extension: insurance.json）

### 寿险核心

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 内含价值 EV 增长率 | ≥ 10% | ≤ 5% |
| 新业务价值 NBV 增长率 | ≥ 10% | 持续负增长 |
| 新业务价值率 VONB | ≥ 30%（友邦级） | ≤ 15% |
| 13 月继续率 / 25 月继续率 | ≥ 90% / ≥ 80% | < 80% / < 70% |
| 投资收益率（综合） | ≥ 5% | ≤ 3% |

### 财险核心

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 综合成本率 COR | ≤ 95%（承保盈利） | ≥ 100%（承保亏损） |
| 综合赔付率 | ≤ 65% | ≥ 75% |
| 综合费用率 | ≤ 35% | ≥ 40% |

### 通用

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 偿付能力充足率（核心 / 综合） | ≥ 150% / ≥ 250% | ≤ 100% / ≤ 150%（监管红线） |
| ROE | ≥ 15% | ≤ 8% |
| 浮存金成本 | < 长期国债收益率 | > 长期国债收益率 |
| 保费收入增长率 | ≥ 8% | ≤ 3% |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1（营收增速）— 应改用保费 / NBV；F3-F5（毛利率 / 库存）— N/A；F9（经营现金流）— 保险现金流定义不同；F12（资产负债率）— 保险天然高负债；F22（存货周转）— N/A
- **改读项**：
  - F7（ROIC）：用 ROE + 投资收益率替代
  - F10（杠杆）：用偿付能力充足率与监管比较
  - F8（应收账款）：应收保费规模可关注，但非核心
- **新增类型专属红旗**：
  - IN1：综合成本率 COR > 100% 连续 2 年（财险承保亏损）
  - IN2：偿付能力充足率距监管红线 < 30 pct
  - IN3：投资资产中地产 / 高风险非标占比 > 30%
  - IN4：新业务价值率 VONB 连续下滑（产品结构恶化）
  - IN5：代理人 / 渠道大幅流失（如 2021-2024 中资寿险代理人腰斩）
  - IN6：长端利率快速下行导致准备金大幅追加
  - IN7：重大灾害（飓风、地震、洪水）+ 再保险覆盖不足

## 六、估值方法优先级

1. **P/EV（内含价值估值）** — 寿险核心方法。EV + 未来 NBV × Multiplier
2. **P/B + ROE** — 财险参考方法
3. **DDM 股息折现** — 高分红保险（中国人寿、平安 H 股）
4. **SOTP** — 综合金融集团（平安：寿险 + 财险 + 银行 + 资管 + 科技）
5. **Float-based valuation** — 浮存金 × 长期投资回报率（巴菲特对 BRK 的内部估值）

关键假设：
- 寿险贴现率（10-11%）
- 长期投资回报率假设（中资寿险通常 5%；近年下调至 4.5-5%）
- 财险 COR 长期均值

## 七、同业对比策略

- **同业范围**：分类（寿险 = 平安 vs 友邦 vs 保诚；财险 = PICC vs Progressive vs Travelers）
- **同业数量**：3-5 家
- **对比维度**：EV / NBV 增长、VONB 率、COR、偿付能力、投资收益率

## 八、关键风险与黑天鹅

1. 长端利率持续下行（准备金压力 + 投资收益缩水）
2. 投资资产暴雷（地产敞口、信用债违约）
3. 重大自然灾害（巨灾财险赔付）
4. 监管政策（报行合一、佣金率限制、产品停售）
5. 代理人 / 渠道流失（寿险销售模式变革）
6. 偿付能力新规（C-ROSS、Solvency II）冲击
7. 长寿风险（年金产品负债延长）
8. 健康险欺诈 / 损失率超预期

## 九、required_pct 默认值

`base = 25%`（V2_SPEC §六）

调整：
- 优秀（友邦、Progressive、BRK）：25% - 5% + 3 = **23%**
- 中资寿险（投资资产暴雷风险）：25% + 0 + 3 = **28%**
- 困境（地产暴雷波及）：+10 = **38%+**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（EV / NBV / 准备金充足性测试）
- 主要数据源 2：监管披露（银保监会 / NAIC / IRDAI 偿付能力报告）
- 主要数据源 3：精算师报告（Embedded Value Report，每年单独发布）
- 主要数据源 4：再保险公告（Munich Re / Swiss Re 数据）
- 主要数据源 5：灾害事件追踪（巨灾、流行病）

## 十一、典型案例参考

**中国平安 601318.SH / 2318.HK**：综合金融集团（寿险 + 财险 + 银行 + 资管 + 科技 + 医疗）。寿险 NBV 受代理人改革、产品结构调整冲击 2021-2024 持续下行。投资组合中房地产敞口（华夏幸福、碧桂园、宝能等）历史问题暴露。**必须用 SOTP**，将寿险 P/EV + 平安银行 PB + 财险 PB + 平安好医生 / 陆金所 等合并估值。叠加 `[a_h_dual_listed, regulated_heavily, creator_led]`。

**Progressive PGR**：美国汽车财险龙头，数据驱动定价（UBI），COR 长期 < 92%，组合 ROE 25%+。可作财险 great_company 候选。
