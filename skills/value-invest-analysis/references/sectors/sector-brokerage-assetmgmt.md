# Sector: 券商/资管 (`brokerage_assetmgmt`)

**典型公司**：
- A 股：中信证券 600030.SH、华泰证券、东方财富、广发证券
- 港股：中金公司 3908.HK、中信证券 6030.HK
- 美股：BlackRock BLK、Charles Schwab SCHW、Goldman Sachs GS、Morgan Stanley MS、KKR KKR、Apollo APO

**v2 schema_extension**：无（base schema；可考虑后续补 AUM / Fee Rate 字段）
**主流派路由**：巴菲特（贝莱德、嘉信） / 张磊（高瓴投资视角）

**默认 modifier_flags**：必带 `regulated_heavily`；中资可叠 `state_owned`

## 一、公司原型识别信号

- 主营经纪佣金 / 投行业务 / 资产管理 / 自营投资
- 收入与资本市场景气度高度相关（贝塔属性强）
- AUM / 客户资产规模是核心
- ROE 周期性大（牛市 20%+，熊市 5-10%）
- 杠杆倍数 5-10×
- 资管型机构（BLK / 高盛资管）收入更稳定（管理费 + 业绩费）

## 二、商业模式六维适配

- **价值主张**：交易中介 + 投融资服务 + 资产配置
- **收入结构**：经纪佣金 + 投行收入 + 资管费 + 自营投资 + 利息净收入
- **成本结构**：人力（高薪）+ IT + 监管合规 + 资本金占用
- **客户结构**：散户 + 机构 + 企业；机构和高净值客户单位价值更高
- **再投资循环**：资本扩张 / 海外并购 / 数字化平台投入
- **关键资源**：客户 AUM / 投行团队 / 资管业绩 / 牌照

## 三、护城河重点

最适用：**规模经济 + 客户切换成本 + 品牌 + 牌照**

注意：相较银行 / 保险，券商护城河相对较弱，多数证券公司可比性强（除资管类巨头）。

锚点：
- 10 分：（罕见）BlackRock（管理费率低但 AUM 万亿 + iShares ETF 网络效应）
- 7 分：高盛投行 + 资管、嘉信平台
- 5 分：中信证券（中国龙头但同质化竞争）
- 3 分：尾部中小券商（无差异化）

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| AUM 增长率 | ≥ 15% | ≤ 5% |
| AUM 净流入（不含市场变化） | 正增长且 > 行业 | 净流出 |
| 平均管理费率（基点） | 稳定（被动 ETF 5-15bps；主动 50-100bps） | 持续下降（费率战） |
| ROE | ≥ 15%（牛市可 25%+） | ≤ 8%（熊市） |
| 业务收入结构 | 资管 / 财富管理 / 海外占比上升 | 自营 + 经纪占比 > 60%（贝塔过强） |
| 经纪业务市占率 | 稳定或上升 | 持续下降 |
| 投行排名 | Top 5（中国 CDR 项目 / 美股 IPO 主承销） | 下降 |
| 自营投资收益率 | ≥ 5% 且波动可控 | 大幅亏损或剧烈波动 |
| 杠杆倍数 | 5-7× | > 10× |
| 信用风控（融资融券担保比例） | ≥ 300% | ≤ 200% |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F1（营收增速）— 与市场周期相关，非线性；F3-F5（毛利率 / 库存）— N/A；F9（经营现金流定义不同）；F12（资产负债率）— 金融机构属性；F22（存货周转）— N/A
- **改读项**：
  - F7（ROIC 低）：用 ROE 替代
  - F10（杠杆）：与监管 + 行业平均比较
  - F8（应收账款）：保证金 / 融出资金 / 自营浮亏需关注
- **新增类型专属红旗**：
  - BR1：AUM 净流出连续 ≥ 2 季度
  - BR2：自营业务连续亏损或巨亏（暴露风控失败）
  - BR3：投行业务被监管处罚 / 暂停资格
  - BR4：股票质押 / 融资融券爆仓（信用减值）
  - BR5：股权激励 / 高管薪酬激增但业绩不佳
  - BR6：监管处罚（内幕交易、操纵市场、虚假陈述）

## 六、估值方法优先级

1. **PB-ROE 估值** — 主方法，类似银行
2. **AUM-based valuation** — 资管类公司用 AUM × 长期 take rate
3. **PE 周期估值带** — 牛熊周期不同 PE 区间
4. **SOTP** — 多业务集团（GS = 投行 + 交易 + 资管 + 财富管理）
5. **DCF** — 资管型公司（BLK / KKR）可用，纯券商不适用

关键假设：
- 周期识别（PE 在不同周期差异极大）
- AUM × 费率 × 利润率
- Ke 10-12%

## 七、同业对比策略

- **同业范围**：分层（被动资管 = BLK vs Vanguard vs SSGA；投行 = GS vs MS vs JPM；中资综合券商 = 中信 vs 中金 vs 国君）
- **同业数量**：3-5 家
- **对比维度**：AUM 增长、管理费率、ROE、自营占比、海外占比

## 八、关键风险与黑天鹅

1. 资本市场熊市持续（自营亏损 + 经纪缩量 + 投行停摆）
2. 监管整顿（A 股 IPO 暂停、香港 IPO 冷淡、SEC 调查）
3. 自营 / 投资爆仓
4. 费率持续下降（嘉信 / 富达零佣金冲击）
5. 客户大规模流失（嘉信收购 TD Ameritrade 整合不顺）
6. 监管资本要求提升
7. 内幕交易 / 操纵市场重大案件
8. ETF 业务被动管理颠覆主动管理（持续 20 年趋势）

## 九、required_pct 默认值

`base = 35%`（V2_SPEC §六，反映周期性 + 同质化）

调整：
- BLK / 嘉信（资管护城河强）：35% - 5% + 3 = **33%**
- 普通中资券商：35% + 0 + 3 = **38%**
- 困境（IPO 暂停 + 熊市）：+10 = **45%+**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（业务分部）
- 主要数据源 2：监管披露（证监会 / SEC / FINRA）
- 主要数据源 3：行业排名（Dealogic / Refinitiv 投行排名）
- 主要数据源 4：ETF.com / 同花顺 ETF 数据库（被动资管）
- 主要数据源 5：万得 AUM 数据库

## 十一、典型案例参考

**BlackRock BLK**：全球最大资管公司，AUM 超 10 万亿美元，iShares ETF 业务占行业 ~40% 份额。被动管理大趋势 + 科技平台（Aladdin）驱动长期成长。可作资管类 great_company 候选（部分时段满足要求）。

**中信证券 600030.SH**：中国头部综合券商，业务覆盖经纪 + 投行 + 资管 + 自营 + 国际化。2024 在 A 股 IPO 暂停 + 熊市背景下 ROE 跌至 8% 左右，PE / PB 见多年低位。叠加 `[a_h_dual_listed, state_owned, regulated_heavily]`。
