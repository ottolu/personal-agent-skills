# Sector: 新能源车/工业周期成长 (`cyclical_industrial_ev`)

**典型公司**：
- A 股：比亚迪 002594.SZ、宁德时代 300750.SZ、隆基绿能、阳光电源、三一重工
- 港股：比亚迪 1211.HK、长城汽车 2333.HK
- 美股：Tesla TSLA、GM、Ford、Caterpillar CAT、Deere DE

**v2 schema_extension**：无（base schema；后续可补 unit economics / battery cost / capacity utilization）
**主流派路由**：段永平（比亚迪长期主义） / 张磊（高瓴新能源投资）

**默认 modifier_flags**：常见 `creator_led`（王传福、马斯克）、`a_h_dual_listed`、`with_policy_support`（新能源补贴 / 碳中和）

## 一、公司原型识别信号

- 主营汽车 / 电池 / 工程机械 / 风电光伏设备 / 半导体设备
- 营收增长强势（≥ 15% 多年），但周期性明显（产能扩张 + 价格战）
- 资本密集（PP&E ≥ 40% 总资产）
- 单位经济模型重要（每辆车毛利 / 每 GWh 电池成本）
- 技术演进快（电池能量密度 / 半导体制程）
- 全球化竞争（出口 + 海外建厂）

## 二、商业模式六维适配

- **价值主张**：替代燃油车 / 传统能源 / 传统工业品；技术升级 + 成本下降双驱动
- **收入结构**：产品销售（车 / 电池 / 设备）+ 部分服务（电池租赁 / 充电 / 软件订阅）
- **成本结构**：BOM + 制造（折旧 + 人力 + 能源）+ 研发 + 销售；规模 + 经验曲线决定 ROIC
- **客户结构**：消费者（C 端车）+ 车企（B 端电池）+ 工业客户（设备）
- **再投资循环**：扩产 + 研发 + 海外建厂；资本开支强度大
- **关键资源**：成本曲线位置 / 技术领先 / 渠道 / 品牌 / 产能规模

## 三、护城河重点

最适用：**规模经济 + 成本曲线 + 技术领先 + 品牌（部分）**

注意：相较于消费品 / 平台，工业周期股护城河相对脆弱（技术变革快、价格战频繁）。

锚点：
- 10 分：（极罕见）特斯拉早期（FSD + 直营 + 电池一体化）、宁德时代峰值（动力电池全球 #1）
- 7 分：比亚迪（垂直一体化 + 王传福 + DM-i 油电混技术）、Caterpillar（全球工程机械 + 售后服务）
- 5 分：其他二三线车企 / 电池厂
- 3 分：尾部产能 / 即将被淘汰的技术路线

验证：
- 单位成本是否持续下降（比亚迪 DM-i 成本曲线领先 5 年）
- 毛利率与同业比较
- 海外份额是否提升

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 营收增长率 | ≥ 30%（成长期）/ ≥ 10%（成熟期） | < 5%（行业放缓） |
| 毛利率 | ≥ 20%（车）/ ≥ 25%（电池） | < 10% / < 15% |
| 单位变动成本 | 成本曲线最左侧 | 持续高于龙头 ≥ 10% |
| ROIC | ≥ 15% | < 10% |
| ROE | ≥ 15% | < 10% |
| Net Debt / EBITDA | ≤ 1.5× | ≥ 3× |
| 产能利用率 | 80-95% | < 60%（过剩）或 > 100%（瓶颈）|
| 库存周转天数 | < 90 | > 150 |
| 研发 / 营收 | 5-10% | < 3% 或 > 15%（效率存疑） |
| 海外收入占比 | ≥ 30%（全球化健康） | < 10%（依赖国内）|
| Capex / OCF | ≤ 80%（扩张期可高）| > 150%（无序扩张） |
| 经营性现金流 / 净利润 | ≥ 1.0 | < 0.8 |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F22（存货周转过快）— 制造业 60-120 天属正常
- **改读项**：
  - F1（营收增速）：必须与行业 / 龙头对比，单纯增长不够
  - F4（毛利率下降）：可能是行业价格战，需对比同业是否同步下降
  - F8（应收账款飙升）：To-B 业务（电池厂、设备）应收账款规模本身大，但增速远超营收增速则警报
  - F10（高杠杆）：扩产期杠杆上升可以但需关注利息覆盖
- **新增类型专属红旗**：
  - CI1：行业产能过剩（如 2024 中国新能源车产能 > 销量 40%）
  - CI2：价格战导致全行业毛利率下降 ≥ 5 pct
  - CI3：技术路线变革风险（如固态电池颠覆液态电池）
  - CI4：补贴退坡 / 关税壁垒（欧盟反补贴关税、美国 IRA 限制中国产品）
  - CI5：单位经济模型恶化（每辆车利润持续下降）
  - CI6：海外建厂受阻（特斯拉柏林、比亚迪欧洲）
  - CI7：核心人物风险（马斯克、王传福、曾毓群）
  - CI8：重大召回 / 安全事故（特斯拉自动驾驶事故、电动车自燃）

## 六、估值方法优先级

1. **DCF（三情景）** — Base / Bull / Bear 主方法
2. **EV/Sales 倍数法** — 高增长期主要参考
3. **PEG** — 增长期辅助
4. **Normalized PE** — 进入成熟期后用穿越周期均值
5. **SOTP** — 多业务集团（比亚迪 = 整车 + 电池 + 半导体 + 海外 + 储能）

关键假设：
- 永续增长率 2-3%
- 周期 + 成长双重不确定性，必须做敏感性分析
- WACC 9-11%

## 七、同业对比策略

- **同业范围**：全球（车企 = 比亚迪 vs Tesla vs Toyota；电池 = CATL vs LG vs Panasonic；设备 = Caterpillar vs Komatsu）
- **同业数量**：3-5 家
- **对比维度**：ROIC、毛利率、单位成本、海外份额、研发强度、产能利用率

## 八、关键风险与黑天鹅

1. 产能过剩导致全行业价格战
2. 技术路线被颠覆（半导体制程、电池化学）
3. 补贴 / 政策退坡
4. 关税 + 贸易战（欧盟 / 美国对中国新能源关税）
5. 全球需求放缓（汽车销量 / 工程机械）
6. 海外建厂 / 出口受阻
7. 核心人物 / CEO 风险
8. 重大召回 / 安全事故 / 自动驾驶事故法律责任
9. 上游原材料价格冲击（碳酸锂、铜、铝）

## 九、required_pct 默认值

`base = 35%`（V2_SPEC §六，反映周期 + 成长双重风险）

调整：
- 优秀（比亚迪、CATL 峰值）：35% - 5% = **30%**
- 行业产能过剩 / 价格战：+10 = **45%**
- 含 distressed（如 Tesla 2025 万一）：+10 = **45%+**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（产销量 / 产能 / 单位成本）
- 主要数据源 2：行业协会（中汽协、CATL 装机量月报、SNE Research）
- 主要数据源 3：海关 / 出口数据
- 主要数据源 4：商品价格（碳酸锂、铜、铝期货）
- 主要数据源 5：监管（欧盟反补贴调查、美国 IRA / IEEPA 公告）
