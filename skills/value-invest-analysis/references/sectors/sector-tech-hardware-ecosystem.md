# Sector: 硬件生态 (`tech_hardware_ecosystem`)

**典型公司**：
- A 股：小米集团（兼 high_growth）
- 港股：小米 1810.HK
- 美股：苹果 AAPL、戴森（未上市）、特斯拉（部分时段，但主类别归 cyclical_industrial_ev）

**v2 schema_extension**：无（base schema）
**主流派路由**：巴菲特（晚年苹果重仓） / 段永平（用户体验）

## 一、公司原型识别信号

- 硬件销售 + 软件 / 服务收入双轮驱动
- 服务收入占比 ≥ 15% 且持续上升
- 用户切换成本来自生态系统（App Store / 配件 / 数据 / 习惯）
- 毛利率结构：硬件 30-40%，服务 60%+
- 拥有 ≥ 5 亿活跃设备装机量（苹果规模）或重要垂直市场份额（戴森吸尘器）
- 软件 + 硬件协同设计能力（垂直整合）

## 二、商业模式六维适配

- **价值主张**：硬件 + 软件 + 服务三位一体体验，超越纯硬件 / 纯软件
- **收入结构**：硬件（一次性 + 周期性升级）+ 服务（订阅 + 抽成）+ 配件
- **成本结构**：硬件 BOM + 渠道 + R&D；规模效应明显
- **客户结构**：消费者 + 商家（App Store 开发者）双边
- **再投资循环**：R&D + 资本开支 + 全球供应链；近年大量回购
- **关键资源**：装机量 / 操作系统 / App 生态 / 自研芯片 / 品牌

## 三、护城河重点

最适用：**生态系统切换成本 + 品牌 + 规模经济 + 自研芯片技术壁垒**

锚点：
- 10 分：苹果（iOS + iCloud + Apple Pay + iMessage + 自研 M 系列芯片 + Apple Watch 健康数据等多重锁定）
- 7 分：小米（IoT 互联生态 + 米粉粘性）

验证：
- 用户切换 Android 的真实样本（极少 / 多年后回归）
- 服务收入毛利率持续 > 60%
- ARPU 持续上升

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 服务收入占比 | ≥ 20% 且上升 | 持续 < 10% |
| 服务收入增长率 | ≥ 15% | < 8% |
| 硬件毛利率 | ≥ 35%（苹果） / ≥ 12%（小米） | 持续下降 |
| 服务毛利率 | ≥ 65% | < 50% |
| 装机量 / 活跃设备数 | 持续增长 | 停滞或下降 |
| ASP（单价） | 稳定或提升 | 持续下降（红海化） |
| R&D / 营收 | 6-10%（持续投入） | > 15%（效率存疑） / < 4%（投入不足） |
| FCF / 营收 | ≥ 20% | < 10% |
| 净现金 + 短投 | 大规模净现金（苹果千亿+） | 净负债且增加 |
| 回购金额 / 净利润 | ≥ 80%（苹果型） | < 30% 同时无并购 |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F10（杠杆）— 此类多巨额净现金；F22（存货周转过快也是问题，需反向看）
- **改读项**：
  - F22 存货周转：硬件公司存货 30-60 天合理；若新品发布前激增可接受
  - F14 SBC：苹果 SBC < 5% 极健康；小米 SBC 较高需检视
  - F26 会计准则变更：关注服务收入确认方式（订阅 vs 一次性）
- **新增类型专属红旗**：
  - TH1：核心硬件市场份额连续 2 年下滑（如 iPhone 在中国份额）
  - TH2：服务增长被监管 / 反垄断打压（App Store 30% 抽成被欧盟 / DMA 限制）
  - TH3：新品类创新缺位（眼镜 / 元宇宙等新硬件平台未能复制 iPhone 成功）
  - TH4：供应链单点依赖暴露（富士康停产事件 / 台积电地缘风险）
  - TH5：自研芯片进展不及预期（Mac M 系列竞争力 / AI 芯片差距）

## 六、估值方法优先级

1. **DCF（10 年 + 永续）** — 主方法
2. **SOTP（硬件 vs 服务分部估值）** — 服务部分给软件估值倍数（PE 25-30×）
3. **EV/EBITDA / EV/Sales** — 苹果通常给 PE 25-30×（高于传统硬件）
4. **Reverse DCF** — 检验当前隐含增长

关键假设：
- 服务增速比硬件高 5-10 pct
- 永续增长率 2-3%
- 别忘了硬件周期性下行风险

## 七、同业对比策略

- **同业范围**：全球（苹果 vs 三星 vs 小米 vs 华为；硬件 = 戴森 vs 索尼）
- **同业数量**：3-5 家
- **对比维度**：服务收入占比、装机量、ARPU、ASP、自研芯片进展

## 八、关键风险与黑天鹅

1. 服务收入抽成被监管下调（App Store 反垄断）
2. 中国市场份额流失（被华为 / 小米 / OPPO 替代）
3. 新平台变革（AR/VR 眼镜失败、AI 硬件颠覆）
4. 关键人物风险（库克接班人 / 设计灵魂人物）
5. 供应链地缘政治（台积电 / 富士康 / 印度）
6. 关税战（关税转嫁能力存疑）
7. 电池 / 显示等核心组件创新停滞
8. 隐私 / AI 监管影响服务能力

## 十、数据源补充

## 十一、典型案例参考

**苹果 AAPL**：iPhone + iOS + App Store + 服务（iCloud / Apple Music / Apple TV+ / Apple Pay）+ Mac + Apple Watch 健康数据生态。巴菲特持仓占伯克希尔市值 ~40%。服务收入毛利率 70%+，2024 年服务占总收入约 25%。风险：服务被反垄断（DMA、App Store 抽成）、Vision Pro 失败、AI 落后。

**小米 1810.HK**：硬件性价比 + IoT 生态 + 互联网服务（广告、游戏）三轮驱动。服务收入占比仍较低（~10%），毛利率结构使其偏向硬件公司。可叠加 `[high_growth_platform]` modifier 解读早期阶段。
