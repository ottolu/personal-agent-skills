# Sector: 科技平台/网络效应 (`tech_platform_network`)

**典型公司**：
- A 股：无典型（A 股科技平台稀缺）
- 港股：腾讯 0700.HK、美团 3690.HK（部分时段更偏 high_growth）
- 美股：Meta META、Alphabet GOOGL、Visa V、Mastercard MA、Booking BKNG、Airbnb ABNB

**v2 schema_extension**：无（使用 base schema）
**主流派路由**：段永平（用户体验 + 商业模式）/ 张磊（动态护城河）

## 一、公司原型识别信号

- 业务为双边或多边平台（用户 + 商家 / 内容生产者 + 消费者 / 持卡人 + 商户）
- 网络效应明显：用户数 / 商家数指数增长后单位成本边际下降
- 高毛利（≥ 60%）、轻资产（PP&E / 总资产 < 20%）
- ROIC ≥ 25%（成熟期），增长期可低于但护城河变宽
- 切换成本明显（已锁定行业标准 / 习惯依赖）
- 数据飞轮（用户行为 → 推荐改进 → 留存提升）

## 二、商业模式六维适配

- **价值主张**：解决信息匹配 / 信任 / 流动性 / 协调成本
- **收入结构**：广告 / 佣金抽成 / 订阅 / 增值服务，多元混合
- **成本结构**：研发 + 服务器 + SBC 是主要 OPEX；CAC 随网络效应递减
- **客户结构**：双边市场，关注 DAU/MAU、ARPU、Take Rate、Retention
- **再投资循环**：研发 + 收购 + AI/算力 资本开支，需高度关注资本配置纪律
- **关键资源**：用户数据 / 网络规模 / 内容生态 / 创始人愿景

## 三、护城河重点

最适用：**网络效应 + 切换成本 + 数据飞轮 三重组合**

锚点示例：
- 10 分：腾讯（社交 + 支付 + 内容生态多重网络互锁）、Visa/MA（双寡头 + 商户/持卡人正反馈）
- 7 分：Meta（社交网络效应面对 TikTok 受挤压但仍宽）、Booking（OTA 网络效应）

验证：
- 单 MAU/DAU 留存率（同行最高 → 网络效应最强）
- Take Rate 是否能维持或提升
- 用户切换至竞品的实际样本（如 Twitter 移民到 Threads / Bluesky 的回归率）

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 营收增长率（YoY） | ≥ 15%（成熟期）/ ≥ 30%（成长期） | < 10% 且 take rate 持平 |
| 毛利率 | ≥ 70% | 持续下降 |
| 运营利润率（剔除 SBC） | ≥ 30% | < 15% |
| Free Cash Flow Margin | ≥ 25% | < 15% |
| SBC / Revenue | ≤ 8% | > 15%（稀释严重） |
| ROIC | ≥ 25% | < 15% |
| DAU/MAU 比例 | ≥ 60% | < 40%（粘性弱） |
| Take Rate / ARPU | 稳定或提升 | 持续下降 |
| 用户留存率 12 月 | ≥ 80% | < 60% |
| 研发费用 / 营收 | 15-25%（合理投入） | > 35%（效率存疑） |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F10（杠杆经营）— 平台多净现金；F22（高存货周转风险）— 无存货
- **改读项**：
  - F14（SBC 高）：必须严格审视。SBC ≥ 15% 视为重红旗（无视 GAAP 调整），稀释股东价值
  - F8（应收/营收异常）：广告平台预付居多，反向考察
  - F11（关联交易）：腾讯系投资生态需特别审视少数股东权益变化
- **新增类型专属红旗**：
  - TP1：MAU/DAU 见顶且 take rate 已极限 → 增长天花板
  - TP2：监管反垄断罚款 / 数据合规调查（如 GDPR / 反垄断）
  - TP3：核心产品被竞品在年轻群体超越（TikTok vs Meta、Bluesky vs X）
  - TP4：创始人 / CEO 离任风险（创始人主导型平台更敏感）
  - TP5：AI / 模型竞赛中明显落后（谷歌搜索 vs ChatGPT 的存在性威胁）

## 六、估值方法优先级

1. **DCF（10 年 + 永续）** — 主方法，永续增长率 3-4%
2. **EV/Sales 或 EV/EBITDA 倍数法** — 成长期平台主要参考
3. **SOTP（分部估值）** — 多业务线公司（腾讯 = 游戏 + 社交 + 金融科技 + 投资）必须 SOTP
4. **Reverse DCF** — 检验市场隐含增长率假设是否过高
5. **PEG / PS-G** — 高增长期间辅助

关键假设：
- 不调整 SBC 不计入费用的 non-GAAP 利润，要 GAAP 视角
- 永续增长率 ≤ 4%
- WACC 8-10%

## 七、同业对比策略

- **同业范围**：全球（社交 = Meta vs 腾讯 vs Snap；广告 = Google vs Meta；支付 = Visa vs MA vs PayPal）
- **同业数量**：3-5 家
- **对比维度**：ROIC、ARPU、用户留存、take rate、研发效率（每元研发对应营收增量）

## 八、关键风险与黑天鹅

1. 监管反垄断（拆分 / 限制并购 / 罚款）
2. 新平台技术变革（短视频替代图文社交、AI agent 替代搜索）
3. 数据合规与隐私（GDPR/CCPA/中国《个人信息保护法》）
4. 货币化模式被颠覆（广告被 AI 屏蔽 / 替代）
5. 创始人 / CEO 重大风险
6. 中概股 VIE / PCAOB 风险（如适用）
7. 海外市场政治限制（TikTok 在美 / 谷歌在中）
8. 内部反复无效的并购（雅虎、Facebook Libra）

## 九、required_pct 默认值

`base = 30%`（V2_SPEC §六，比 great_company 略高，因技术变革风险）

调整后典型范围：
- 高质量平台（腾讯、Visa）：30% - 5% = **25%**
- 多 modifier（VIE + PCAOB）：30% + 5 + 5 = **40%**

## 十、数据源补充

- 主要数据源 1：年报 10-K（用户运营数据 + 业务分部）
- 主要数据源 2：季度 SBC 与 GAAP 利润对比
- 主要数据源 3：第三方监测（App Annie / Sensor Tower / 极光、QuestMobile）
- 主要数据源 4：监管申请 / DOJ / SAMR 反垄断公告

## 十一、典型案例参考

**腾讯 0700.HK**：游戏 + 社交（微信 + QQ）+ 金融科技 + 投资组合。多重护城河（社交网络效应 + 切换成本 + 内容生态 + 投资飞轮）。须用 SOTP 估值。风险：游戏版号、未成年人保护、宏观广告周期、东南亚游戏出海竞争。

**Visa V**：四方支付网络的双寡头之一，take rate 0.13% × 万亿美元交易量。极高 ROIC（30%+）。风险：监管降费、本地央行清算系统（中国银联、巴西 Pix）替代、加密支付。
