# Sector: 成熟制药 (`pharma_mature`)

**典型公司**：
- A 股：恒瑞医药 600276.SH、复星医药（部分）、华东医药
- 港股：石药集团 1093.HK、中国生物制药 1177.HK
- 美股：Pfizer PFE、Johnson & Johnson JNJ、Merck MRK、Eli Lilly LLY、AbbVie ABBV

**v2 schema_extension**：无（base schema；可考虑后续补 patent_cliff / pipeline 字段）
**主流派路由**：巴菲特（曾持 JNJ / Merck） / 张磊（高瓴医药投资）

## 一、公司原型识别信号

- 已盈利 ≥ 5 年，ROE ≥ 10%
- 上市药品组合 ≥ 10 个，主要品种营收占比分散（无单一品种 > 40% 风险）
- R&D / 营收 12-25%（已有商业化基础但持续投入）
- 拥有持续 BD / 并购整合能力
- 主营毛利率 ≥ 60%
- 经营性现金流稳定为正

## 二、商业模式六维适配

- **价值主张**：治疗或缓解多种疾病，覆盖多个治疗领域
- **收入结构**：处方药销售（医院 / 药店）+ OTC + 部分 BD；多元品种降低单点风险
- **成本结构**：R&D + 销售（医药代表）+ 制造 + 合规；销售费用通常高（A 股 30%+）
- **客户结构**：医院 / 药店 / 患者；医保支付方是关键
- **再投资循环**：R&D + BD / 并购扩充管线 + 国际化扩张
- **关键资源**：管线 + 已上市品种 + 销售网络 + 制造质量 + 医保准入

## 三、护城河重点

最适用：**专利 + 监管壁垒 + 销售网络 + 品牌（少数 OTC）**

注意"专利悬崖"是此类核心风险 —— 大品种专利到期后通用名药快速抢占 80% 市场。

锚点：
- 7 分：JNJ（多元品种 + 医疗器械 + 消费保健三轮）、LLY（GLP-1 蒸蒸日上）
- 5 分：Pfizer 后疫情期（COVID 收入断崖 + Ibrance 等专利悬崖）
- 3 分：仿制 / 一致性评价压价的中国大输液 / 大宗仿制药

## 四、核心财务指标

| 指标 | 优秀 | 警戒 |
|---|---|---|
| 营收增长率 | ≥ 8% | < 3% 或负增长 |
| 毛利率 | ≥ 65% | < 50% |
| ROIC | ≥ 15% | < 10% |
| ROE | ≥ 15% | < 10% |
| R&D / 营收 | 12-20%（恒瑞约 15-18%） | < 8% 或 > 25%（效率存疑） |
| 销售费用 / 营收（A 股） | 25-35%（行业平均） | > 40% |
| 管线 NPV / 市值 | ≥ 30% | < 10%（管线匮乏） |
| 经营性现金流 / 净利润 | ≥ 1.0 | < 0.8 |
| 应收账款周转天数 | < 90 | > 120 |
| Top 3 品种营收占比 | < 50%（分散健康） | > 70%（集中风险） |
| 专利悬崖 5 年内大品种占比 | < 30% | > 50% |

## 五、红旗清单 N/A / 改读规则

- **N/A 项**：F22（存货周转过慢）— 制药存货 90-180 天属正常
- **改读项**：
  - F1（营收增速 ≪ 行业）：制药行业增速本身受集采 / 一致性评价压制，需对比国际同业
  - F4（毛利率下降）：可能因医保集采降价，要看是否系单一品种还是整体
  - F8（应收账款）：医院回款慢是普遍现象，但若 > 180 天需追问
- **新增类型专属红旗**：
  - PM1：大品种专利 < 3 年到期且无后继管线（专利悬崖）
  - PM2：医保集采 / 国谈大幅降价（A 股恒瑞、石药案例）
  - PM3：FDA / NMPA 警告信 / 暂停销售
  - PM4：销售费用率突然飙升（带金销售合规问题）
  - PM5：大额商誉减值（并购整合失败，如部分中国药企海外资产减值）
  - PM6：主要品种被纳入"集采高风险品种"清单

## 六、估值方法优先级

1. **DCF + Pipeline rNPV** — 已上市品种 DCF + 在研管线 rNPV 加总
2. **PE 历史估值带** — 同业相对估值
3. **Sum-of-Parts** — 多业务线（处方 + OTC + 医疗器械 + 消费保健）
4. **EV/EBITDA** — 跨国可比

关键假设：
- 永续增长率 2-3%
- 大品种 5-10 年现金流 + 后续大幅折扣
- 医保集采假设（中国市场降价 50-90% 是常态）

## 七、同业对比策略

- **同业范围**：跨市场（恒瑞 vs LLY vs Roche vs Novartis；治疗领域细分对比）
- **同业数量**：3-5 家国际龙头
- **对比维度**：管线丰富度、专利悬崖、ROIC、国际收入占比、R&D 效率

## 八、关键风险与黑天鹅

1. 集采 / 一致性评价（中国市场）
2. 美国 IRA 药价谈判（针对 Medicare 高销售品种）
3. 专利悬崖 + 仿制冲击
4. FDA / NMPA / EMA 监管警告 / 撤市
5. 重大产品安全事件（黑框警告、患者死亡）
6. 销售合规问题（带金销售调查）
7. BD / 并购整合失败导致商誉减值
8. 关键产品被新一代疗法颠覆（小分子被生物药 / 细胞疗法替代）

## 九、required_pct 默认值

`base = 30%`（V2_SPEC §六）

调整：
- 优秀（LLY GLP-1 时代）：30% - 5% = **25%**
- 集采承压（恒瑞）：30% + 0 = **30%**
- 状态低迷（Pfizer 后疫情）：+10 = **40%**

## 十、数据源补充

- 主要数据源 1：年报 + 季报（品种销售拆分）
- 主要数据源 2：FDA Orange Book / 国家药监局数据库（专利到期）
- 主要数据源 3：医保局集采公告 / 国谈结果
- 主要数据源 4：IQVIA / 米内网 销售数据
- 主要数据源 5：临床试验注册数据库（管线进度）

## 十一、典型案例参考

**Eli Lilly LLY**：GLP-1（Mounjaro / Zepbound）爆款驱动 2023-2024 估值飞升，ROE 50%+，PE 60×+。管线（Donanemab 阿尔茨海默病、口服 GLP-1）支撑长期增长。风险：制造扩产瓶颈、Medicare 谈判（IRA 法案 2026 起）、新一代肥胖药竞争（Amgen / Novo 后续品种）。

**恒瑞医药 600276.SH**：中国创新药龙头，2018-2021 估值见顶后受集采 / 一致性评价压制大幅回调。2024 创新药管线兑现（PD-1、ADC、KRAS、GLP-1 等），从 pharma_mature 边缘向 great_company 复兴。
